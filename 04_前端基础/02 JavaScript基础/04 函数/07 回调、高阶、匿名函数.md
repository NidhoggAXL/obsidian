```js
function foo(fn) {
  //bar传入foo，在被foo调用bar，就是回调函数(callback function)
  fn()
}

function bar() {
  console.log("bar被执行")
}

foo(bar)//bar被执行
```

foo 这种函数也叫做高阶函数(Higher-order function)


高阶函数**必须至少满足两个条件之一**

 * 接受一个或多个函数作为输入;
* 输出一个函数


匿名(anonymous)函数的理解:

* 如果在传入一个函数时，我们没有指定这个函数的名词或者通过函数表达式指定函数对应的变量，那么这个函数称之为匿名函数。


```js
function foo(fn) {
  //匿名函数作为实参传入foo，形参fn接收
  //foo内部再执行fn函数
  fn()
}

foo(function(){console.log("匿名函数被执行")})//匿名函数被执行
```

在对foo传入函数的时候没有函数名，这种函数就叫做匿名函数
