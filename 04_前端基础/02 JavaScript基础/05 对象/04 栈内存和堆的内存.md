
# 一、栈内存和堆内存
我们知道程序是需要加载到内存中来执行的，我们可以将内存划分为两个区域:**栈内存和堆内存**

* **原始类型**占据的空间是在**栈内存**中分配的;
* **对象类型**占据的空间是在**堆内存**中分配的

# 二、值类型和引用类型

原始类型的保存方式:在变量中保存的是值本身

* 所以原始类型也被称之为值类型

对象类型的保存方式:在变量中保存的是对象的“引用

* 所以对象类型也被称之为引用类型

# 三、思考下面的现象

**现象一：**

```js
var bar1 = {}
var bar2 = {}
console.log(bar1 === bar2)//flase
console.log(bar1 == bar2)//flase
```

两个对象的地址不同，所以为flase

**现象二：引用传递，但是在函数中创建了一个新对象，没有对传入对象进行修改**

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17275316850001gsacq.png)

obj 刚传入函数foo 的时候 a 的地址确实是和 obj 同一个地址，但是在**函数内部的时候 a 有创建了一个对象**，这个时候 a 指向的地址就是另一个新创建的对象，所以 obj 的name没有改变

**现象三：引用传递，但是对传入的对象进行修改**

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1727531890000qyzucn.png)

传入函数内部没有创建对象，并没有改变地址，所以对 obj 的属性进行了改变。
