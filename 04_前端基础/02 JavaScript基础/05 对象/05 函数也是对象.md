```js
var bar1 = function(){}//函数
var bar2 = new CreateObject()//构造函数
var bar3 = {}//对象

console.log(bar1)//function
console.log(bar2)//object
console.log(bar3)//object
```

从上面可以看出来构造函数也是一个对象


**从内存的角度来说：**

函数的执行体和对象的属性都是存储在内存的**堆里面**，函数名和对象名都只是保存（地址-指针-引用）

居然函数也是一个对象，那么我们在函数中使用对象的语法结构可以吗？

当然是可以的：

```js
function foo() {
  console.log("foo")
}

//为foo函数对象添加bar方法
foo.bar = function() {
  console.log("bar")
}
  
foo.bar()//bar
```
