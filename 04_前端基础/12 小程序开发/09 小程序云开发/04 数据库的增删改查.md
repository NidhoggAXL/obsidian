# 三、删除数据

```js
collection.doc('718e0366689497c700f82a5570cd4bff')
  .remove().then(res => console.log(res))
```

这个 id 是在控制台添加的数据，没有权限在 代码 (前端) 进行删除,所有要改变权限后才可以删除.不然会报错误

# 五、查询数据

全部的查询方法:https://developers.weixin.qq.com/miniprogram/dev/wxcloudservice/wxcloud/reference-sdk-api/database/Collection.html

常用的查询方式如下：

**方式一**：通过ID查询精确的某一条数据； 

- 使用doc查询ID

```js
const myTodo = db.collection('todos').doc('my-todo-id')
```

**方式二**：根据条件查询满足条件的数据； 

- 使用where作为条件

```js
const _ = db.command
	const result = db.collection('todos').where({
  price: _.lt(100)//lt小于
}).get()
```

**方式三**：通过指令过滤数据； 

- 使用db.command的查询指令 


**方式四**：通过正则表达式匹配符合的数据； 

- 使用db.RegExp创建正则规则 

```js
// 原生 JavaScript 对象
db.collection('todos').where({
  description: /miniprogram/i
})

// 数据库正则对象
db.collection('todos').where({
  description: db.RegExp({
    regexp: 'miniprogram',
    options: 'i',
  })
})

// 用 new 构造也是可以的
db.collection('todos').where({
  description: new db.RegExp({
    regexp: 'miniprogram',
    options: 'i',
  })
})
```

**方式五**：获取整个集合的数据（<mark class="hltr-orange">小程序端一次性最多20条，云函数中可以获取100条</mark>）； 

- 直接调用get 

**方式六**：过滤、分页、排序查询数据 

- 使用[field](https://developers.weixin.qq.com/miniprogram/dev/wxcloudservice/wxcloud/reference-sdk-api/database/collection/Collection.field.html)(过滤)、[skip](https://developers.weixin.qq.com/miniprogram/dev/wxcloudservice/wxcloud/reference-sdk-api/database/collection/Collection.skip.html)(分页)、[limit](https://developers.weixin.qq.com/miniprogram/dev/wxcloudservice/wxcloud/reference-sdk-api/database/collection/Collection.limit.html)(数量)、[orderBy](https://developers.weixin.qq.com/miniprogram/dev/wxcloudservice/wxcloud/reference-sdk-api/database/collection/Collection.orderBy.html)

## 5.1 查询指令
假设我们需要查询进度大于 30% 的待办事项，那么传入对象表示全等匹配的方式就无法满足了，这时就需要用到查询指令。

数据库 API 提供了大于、小于等多种查询指令，这些指令都暴露在 **db.command** 对象上。
