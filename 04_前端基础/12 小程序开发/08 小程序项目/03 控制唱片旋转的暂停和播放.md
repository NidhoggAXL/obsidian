
先通过 [[07 transform形变|transform]] 和 [[10 animation动画|animation]] 对对话进行一个设置：

```css
@keyframes albumAnim {
  0% {
    transform: rotate(0)
  }
  100% {
    transform: rotate(360deg);
  }
}

.album-anim {
  animation: albumAnim 10s linear infinite;
  /* linear-匀速动画 infinite-无线循环 */
}
```

**如果是为一张图片添加这个动画**：

```html
<image 
  class="{{isPlayre? 'album-anim': '' }}" 
  src="图片URL地址"
/>
```

通过定义一个变量 isPlayer 来控制是否添加 album-anim ，这样就可以进行是否进行 animation 动画了，但是不可以保存单曲的播放帧，移除 class 后就会回到最初的转台。

**解决方案如下**：

```html
<image 
  class="album-anim" 
  src="图片URL地址"
  style="animation-play-state: {{ isPlayer? 'running': 'paused'}};"
/>
  
<!-- animation-play-state样式中
running - 继续运行上一帧的动画
paused - 停止当前帧动画
-->
```

这样通过 isPlayer 来控制的话，不需要进行移除 alibum-anim 动画，而是对动画本事进行一个控制，就可以保存上一帧的运动状态。
