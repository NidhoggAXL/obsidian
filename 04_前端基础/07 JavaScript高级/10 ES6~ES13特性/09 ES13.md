# 二、Object.hasOwn

Object中新增了一个静态方法（类方法）： hasOwn(obj, propKey)

* 该方法用于判断一个对象中是否有某个自己的属性； 

那么和之前学习的Object.prototype.hasOwnProperty有什么区别呢？

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1732181199000o8xbld.png)


**区别一**：防止对象内部有重写hasOwnProperty 

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1732181302000y0zz1e.png)


**区别二**：对于隐式原型指向null的对象， hasOwnProperty无法进行判断

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1732181502000ay1h5k.png)

# 三、New members of classes

在ES13中，新增了定义class类中成员字段（field）的其他方式：

* Instance public fields - 实例公共字段
* Static public fields - 静态公共字段
* Instance private fields - 实例私有字段
* static private fields - 静态私有字段
* static block - 静态块
