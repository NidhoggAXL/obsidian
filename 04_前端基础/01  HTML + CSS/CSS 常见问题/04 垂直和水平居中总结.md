# 一、水平居中
**在一些需求中，需要元素在父元素中水平居中显示(父元素一般都是块级元素、inline-block)**

**行内级元素(包括inline-block元素)**
* 水平居中:在父元素中设置text-align:center

**块级元素**
* 水平居中`margin: 0 auto`

**[[04 绝对定位-absolute|绝对定位]]**

* 元素有**宽度**的情况下

```css
left:0;
right:0;
margin:0 auto;
```

[[09 Flex布局#4.4 justify-content|flex布局]]

* 父级元素设置`justify-content:center;`

# 二、垂直居中

## 2.1 绝对定位解决

**元素有高度的情况下**

```css
top:0;
bottom:0;
margin: auto 0;
```
 
> [!tip] 弊端
> 1. 元素使用定位（脱离[[01 标准流(Normal Flow)布局|标准流]]）
> 2. 必须给元素设置高度[](01%20标准流(Normal%20Flow)布局.md)md)4.6 align-iems|flex]]

* 父级元素设置`align-items: center ;`

> [!tip] 弊端
> 当前 flex 布局中的所有元素都会被垂直居中，不便与对每个元素个体的设置。


## 2.3 top+translate()

1. 让元素向下移**父元素**的 50%
2. 让原子向下移**自身**的 50%

![[垂直居中_transform_translate]]

**思路：**

就是让垂直居中的元素中线低于包含块中线自身元素的50%，在使用 translate 向上移动自身的 50%

 **百分比问题**
 
* 第一步的 `top: 50% ;`是相对与 包含块高度的 50%
	* 为什么使用的是 top 而不是 `margin-top: 50% ;`
	* 因为 margin-top 的百分比是**相对与包含快的宽度**
* 第二的 `transform: translate( 0 , 50% );`是相对与元素本身高度的 50%
