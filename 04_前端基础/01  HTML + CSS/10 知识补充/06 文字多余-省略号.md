# 一、单行文字

```css
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
```

| **white-space** |   **空白区域**   |
| :-------------: | :----------: |
|   **nowrap**    | **禁止文字自动换行** |
|  **overflow**   |  **溢出、泛滥**   |
|  **ellipsis**   |    **省略**    |
|     **hidden**      |      **隐藏**      |

> [!tip] 注意
> 上面的代码只对块级元素可以进行省略号

# 二、多行文字

```css
	overflow: hidden;
	text-overflow: ellipsis;
	display: -webkit-box;
	/* 限制在几行里面 */
	-webkit-line-clamp: 2;
	-webkit-box-orient: vertical;
```


**例如从第二行开始省略：**

```html
<style>
	.contaner {
		width:400px;
		height: 40px;
		background-color: orange;

		overflow: hidden;
		text-overflow: ellipsis;
		display: -webkit-box;
		/* 第几行开始多余的省略 */
		-webkit-line-clamp: 2;
		-webkit-box-orient: vertical;
	}
</style>

<div class="contaner">都说陈平安很苦，可他哪有我苦？我本生于耕农之家，落地时天无异象，父母识字略过百，瘦田刨食，土dcoder风雨,人前却不敢卸下伪装，人后不敢直视内心肮脏</div>
```
