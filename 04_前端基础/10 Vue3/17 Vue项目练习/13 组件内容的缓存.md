在 Vue 项目中进行数据缓存的时候[[04 keep-alive组件|keep-alive]]，
如果是有router路由的时候（router-view)的时候是需要使用如下的使用的：

1. 在App组件里面

```html
<!-- name属性 -->
<router-view v-slot="props">
  <keep-alive include="home">
	<component :is="props.Component"></component>
  </keep-alive>
</router-view>
```

2. 需要给home组件进行配置

> 这个编码要单独放到 script 的代码块里面，不可以和其他代码在同一个代码里面

```js
<script>
export default {
  name: 'home' // 必须与 include 中的名称匹配
}
</script>
```
