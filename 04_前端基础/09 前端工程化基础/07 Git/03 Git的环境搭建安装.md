# 四、Git的配置选项

安装Git后，要做的第一件事就是设置你的用户名和邮件地址。 

* 这一点很重要，因为**每一个 Git 提交都会使用这些信息**，它们会写入到你的每一次提交中，不可更改； 
* 如果使用了 **--global 选项，那么该命令只需要运行一次**，因为之后无论你在该系统上做任何事情， Git 都会使用那些信息；

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1744806113000bl4sdy.png)
