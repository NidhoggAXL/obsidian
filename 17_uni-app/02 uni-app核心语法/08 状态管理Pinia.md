# 一、认识Pinia

Pinia（发音为 /piːnjʌ/，如英语中的 peenya） 是 Vue 的存储库，它允许跨组件、页面共享状态。

**uni-app 内置了 Pinia，使用 HBuilder X 不需要手动安装，直接使用即可。**

使用 CLI 需要手动安装，执行 yarn add pinia 或 npm install pinia。


# 二、使用Pinia
