# 一、Options API

uni-app 常用的页面生命周期函数：

 - onLoad(options) -> onLoad
 - onShow -> onShow
 - onReady -> onReady
 - onHide -> onHide
 - onUnload -> onUnload
 - onPullDownRefresh -> onPullDownRefresh
 - onReachBottom -> onReachBottom
 - 更多： https://uniapp.dcloud.net.cn/tutorial/page.html#lifecycle

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1765374436000iiqh8v.png)

> [!tip] 注意事项： 
>  - 页面可以使用Vue组件生命周期吗？ 可以的 
>  - 页面滚动才会触发 onReachBottom 回调，如果自行通过overflow实现的滚动不会触发 onReachBottom 回调


# 二、Composition API

Composition 中使用生命周期的时候，需要注意的是

 - uni-app 提供的生命周期需要**手动的导入**生命周期
 - Vue 的生命周期会**自动的导入**

```js
import {
	onLoad,
	onShow,
	onUnload,
} from "@dcloudio/uni-app"//手动导入uni-app的页面生命周期
import { onBeforeMount, onMounted } from "vue";//Vue生命周期自动导入

//uni的页面生命周期
onLoad(() => {
	console.log("Vue3 onLoad");
})
onShow(() => {
	console.log("Vue3 onShow");
})

//Vue组件的生命周期
onBeforeMount(() => {
	console.log("Vue3 onBeforeMount");
})
onMounted(() => {
	console.log("Vue3 onMounted");
})

```
