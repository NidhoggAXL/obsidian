# 一、Options API

uni-app 常用的页面生命周期函数：

 - onLoad(options) -> onLoad
 - onShow -> onShow
 - onReady -> onReady
 - onHide -> onHide
 - onUnload -> onUnload
 - onPullDownRefresh -> onPullDownRefresh
 - onReachBottom -> onReachBottom
 - 更多： https://uniapp.dcloud.net.cn/tutorial/page.html#lifecycle

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1765374436000iiqh8v.png)

> [!tip] 注意事项： 
>  - 页面可以使用Vue组件生命周期吗？ 可以的 
>  - 页面滚动才会触发 onReachBottom 回调，如果自行通过overflow实现的滚动不会触发 onReachBottom 回调
