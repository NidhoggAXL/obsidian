# 一、自定义Hook

自定义Hook本质上只是一种函数代码逻辑的抽取，严格意义上来说，它本身并不算React的特性。 

需求：所有的组件在创建和销毁时都进行打印

- 组件被销毁：打印 组件被销毁了；
