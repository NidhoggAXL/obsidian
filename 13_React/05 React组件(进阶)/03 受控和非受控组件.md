# 二、受控组件的基本演练

在 HTML 中，表单元素（如`<input>`、`<texarea>` 和 `<select>`）之类的表单元素通常浏览器自己进行维护 state，并根据用户输入进行更新。

而在 React 中，可变状态（mutable state）通常保存在组件的 **state 属性中**，并且**只能通过使用 setState()来更新**。 

- 将两者结合起来，使React的state成为“**唯一数据源**”； 
- 渲染表单的 **React 组件**还**控制着用户输入过程中表单发生的操作**； 
- 被 React 以这种方式控制取值的表单输入元素就叫做“**受控组件**”；

**受控组件和非受控组件的代码区别：**

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1755246865000lpnt17.png)

于在表单元素上设置了 value 属性，因此显示的值将始终为 this.state.value，这使得 React 的 state 成为唯一数据源。


![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1755246988000sc3l5t.png)


```jsx
import React, { PureComponent } from 'react'

export class App extends PureComponent {
  constructor() {
    super()

    this.state = {
      userName: '',
      password: ''
    }
  }

  // 提交表单的监听
  submitEvent(event) {
    //1.阻止from的自动提交事件
    event.preventDefault()
    
    //2.获取所有表单数据，对数据进行组件
    console.log(this.state.userName, this.state.password)
  } 

  // input输入事件的监听
  handerlinputChange(event) {
    //使用计算属性名
    const inputName = event.target.name 
    // console.log(inputName)
    this.setState({
      [inputName]: event.target.value
    })
  }

  render() {
    const { userName, password } = this.state
    return (
      <div>
        <form onSubmit={e => this.submitEvent(e)}>
          {/* htmlFor就是html的label中的for属性 */}
          <label htmlFor="userName">
            用户：
            <input
              id='userName'
              type="text"
              name='userName'
              value={userName}
              onChange={e => this.handerlinputChange(e)}
            />
          </label>
          <label htmlFor="password">
            密码:
            <input
              id='password'
              type="password"
              name='password'
              value={password}
              onChange={e => this.handerlinputChange(e)}
            />
          </label>
          <button type='submit'>提交</button>
        </form>
      </div>
    )
  }
}

export default App
```

> [!info] 用到的技术
> - 监听form表单的提交，阻止from的自动提交
> - label的htmlFor属性，是防止使用 for 的关键字，和 className 一样的作用

## 3.1 checkbox复选框

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/17552498250000fi4ac.png)

```jsx
import React, { PureComponent } from 'react'

export default class Home extends PureComponent {
  constructor() {
    super()

    this.state = {
      isAgree: true,
      hobbys: [
        { value: 'sing' , text: '唱', isCheck: false },
        { value: 'dance', text: '跳', isCheck: false },
        { value: 'rap', text: 'rap' , isCheck: false }
      ]
    }
  }

  // 单选改变
  changAgree(event) {
    this.setState({ isAgree: event.target.checked })
  }

  // 多选改变
  handelHobby(event, index) {
    //展开语法，保证是一个可以改变里面的对象，
    //而不是拿到一个空数组
    const hobbies = [...this.state.hobbys]
    hobbies[index].isCheck = event.target.checked
    this.setState({ hobbys: hobbies})
  }

  //提交多选
  submtiClick(event) {
    // 阻止默认提交
    event.preventDefault()

    //提交选中的hobbys
    const hobbies = this.state.hobbys.filter(item => item.isCheck).map(item => item.value)
    console.log("我的爱好", hobbies)
  }

  render() {
    const { isAgree, hobbys } = this.state
    return (
      <div>

        <div>
          {/* 单选 */}
          <label htmlFor="hobby">
            <input
              type="checkbox"
              id='hobby'
              checked={isAgree}
              onChange={e => this.changAgree(e)}
            /> 篮球
          </label>
        </div>

        <div>
          {/* 所选 */}
          <div>你的爱好</div>
          <form onSubmit={e => this.submtiClick(e)}>
            {
              hobbys.map((item, index) => {
                return (
                  <label htmlFor={item.value} key={index}>
                    <input
                      id={item.value}
                      type="checkbox"
                      checked={item.isCheck}
                      onChange={e => this.handelHobby(e, index)}
                    /> {item.text}
                  </label>
                )
              })
            }
            <button type='submit'>提交</button>
          </form>
        </div>
      </div>
    )
  }
}

```

## 3.2 select

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1755256069000rpsh5f.png)

```jsx
import React, { PureComponent } from 'react'

export default class Info extends PureComponent {
  constructor() {
    super()
    this.state = {
      fruit: '苹果',
      // fruits: ['苹果', '西瓜', '香蕉']
      fruits: []
    }
  }

  appleChange(event) {
    this.setState({ fruit: event.target.value })
  }

  selecteOptions(event) {
    const options = Array.from(event.target.selectedOptions)
    const values = options.map(item => item.value)
    this.setState({ fruits: values })
  }

  render() {
    const { fruit, fruits } = this.state
    return (
      <div>
        {/* 单选 */}
        <div>
          <h4>单选</h4>
          <select value={fruit} onChange={e => this.appleChange(e)}>
            <option>苹果</option>
            <option>西瓜</option>
            <option>香蕉</option>
          </select>
          <h3>喜欢的睡过是:{fruit}</h3>
        </div>
        {/* 多选 */}
        <div>
          <h4>多选</h4>
          <select value={fruits} multiple onChange={e => this.selecteOptions(e)}>
            <option>苹果</option>
            <option>西瓜</option>
            <option>香蕉</option>
          </select>
        </div>
      </div>
    )
  }
}

```

#  四、非受控组件

React推荐大多数情况下使用 **受控组件** 来处理表单数据:

- 一个**受控组件**中，表单数据是由 **React 组件来管理**的;
- 另一种替代方案是**使用非受控组件**，这时表单数据将**交由 DOM 节点**来处理

如果要使用非受控组件中的数据，那么我们需要使用 ref 来从DOM节点中获取表单数据

在非受控组件中通常使用defaultValue来设置默认值；

```jsx
<input type="text" defaultValue="aoaao" />
```

`同样，<input type="checkbox">和 <input type="radio">支持 defaultchecked,<select>和 <textarea> 支持 defaultValue。`
