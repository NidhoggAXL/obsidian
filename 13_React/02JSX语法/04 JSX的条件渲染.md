某些情况下，界面的内容会根据不同的情况显示不同的内容，或者决定是否渲染某部分内容： 

- 在vue中，我们会通过指令来控制：比如v-if、v-show； 
- 在React中，**所有的条件判断都和普通的JavaScript代码一致**；

**常见的条件渲染的方式有哪些呢？** 

- **方式一**：条件判断语句：适合逻辑较多的情况 

- **方式二**：三元运算符 ：适合逻辑比较简单

- **方式三**：与运算符&& ：适合如果条件成立，渲染某一个组件；如果条件不成立，什么内容也不渲染； 
	- 场景：当某一个值，可能为undeifned的时，使用&&进行条件判断(类似从服务器获取数据，要有值的时候在显式)

- 方式四：或运算符 ||：适合如果条件成立，渲染第一个true的组件；如果条件不成立，渲染默认内容；
	- 场景：默认显示头像，当用户有选择头像的时候，渲染用户选择的头像，如果没有选择，则显示默认头像。

```jsx
<div>
  { friends && <div>{friends.name + " " + friends.desc}</div> }
</div>
```

- **v-show的效果** ：主要是控制display属性是否为none
