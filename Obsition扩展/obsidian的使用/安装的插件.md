# DataView  搜索查询（快捷健查询）
## 搜索
*  搜索技巧
	*  直接搜索关键词
		* 搜索多个关键词的文档（空格间隙）
			* 在两个关键词之间输入大写的 OR（表示且的意思）
			* 有可以输入一个减号表示且的意思
		* 指定搜索范围
			* 搜索文件名 file：word（文件名）
			* 搜索文本内容 content：word
			* 搜索标签 tag：# tag：word
			* 搜索同一行中的多个关键词 line：word1 word2
			* 搜索同一个章节中的多个关键词 section：word1 word2
			* 搜索同一段落（块）中的多个关键词 block：word1 word2
	* 搜索任务
		* 搜索任务 task："" (英文状态下的引号)
		* 搜索为完成的任务 task:todo: ""
		* 搜索未完成的任务 task:done: ""
	* 保存搜索的结构
		* 代码 query
		* 类如：
```query
双链
```
