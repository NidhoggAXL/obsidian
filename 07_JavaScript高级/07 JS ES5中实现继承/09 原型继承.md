通过上面的图解可以得到下面的结论：

* 函数作为函数有显式原型（prototype），作为函数对象也有隐式原型（`__proto__`）
* 对象只有隐式原型（`__proto__`）
* 所有函数的显式原型对象中的隐式原型（`__proto__`）都指向 Object 的显式原型，而 Object 的显式原型对象中的隐式原型（`__proto__`）指向空
* 对象的隐式原型（`__proto__`）都指向构造这个对象函数的显式原型（`prototype`）
* 总上所述，**不管是对象还是函数最后指向都是 Object 函数的的显式原型对象，而显式原型对象的隐式原型为 null ，这样就会最在断链，不会永无止境。**
