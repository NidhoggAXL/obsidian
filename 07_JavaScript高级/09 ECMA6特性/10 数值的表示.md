在ES6中规范了二进制和八进制的写法

```js
<script>
  //十进制表示
  let a = 100
  
  //二进制表示：binary
  let b = 0b100
  
  //八进制表示：octonary
  let c = 0o100
  
  //十六进制表示：hexadecimal
  let d = 0x100
  
  console.log(a, b, c, d)//100 4 64 256
</script>
```

另外在ES2021新增特性:数字过长时，可以使用作为**连接符**

```js
let a = 1000_0000_0000
```
