# 三、Optional Chaining

**可选链**也是ES11中新增一个特性，主要作用是让我们的代码在**进行null和undefined判断时更加清晰和简洁：**

```js
<script>
  const obj = {
    name: "axl",
    friend: {
      running: function() {
        console.log("running~")
      }
    }
  }
  
  obj?.friend?.running?.()//running~
</script>
```

> 我们在调用 obj 里面的方法的时候，其实可能会不确定里面是不是有我们想调用的方法，这个时候就可以使用可选链来进行操作。
