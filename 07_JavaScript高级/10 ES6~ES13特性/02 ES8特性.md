# 一、Object values
之前我们可以通过 Object.keys 获取一个对象所有的key 

在ES8中提供了 Object.values 来获取所有的value值：

```js
<script>
  const obj = {
    name: "axl",
    age: 18
  }
  console.log(Object.keys(obj))//['name', 'age']
  console.log(Object.values(obj))//['axl', 18]
</script>
```

# 二、Object entries

通过 Object.entries 可以获取到一个数组，数组中会存放**可枚举属性的键值对数组**。

* 可以针对**对象**、**数组**、**字符串**进行操作；

```js
<script>
  const obj = {
    name: "axl",
    age: 19
  }
  
  console.log(Object.entries(obj))
  console.log(Object.entries(["mba", "aba", "cba"]))
  console.log(Object.entries("hello"))
</script>
```
