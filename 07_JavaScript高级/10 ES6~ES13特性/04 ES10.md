# 一、flat flatMap

flat() 方法会按照一个可指定的深度递归遍历数组，并将所有元素与遍历到的子数组中的元素合并为一个新数组返回。

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1732169508000fl9mgk.png)


flatMap() 方法首先使用映射函数映射每个元素，然后将结果压缩成一个新数组。 

* 注意一：flatMap是先进行[[09 数据结构-Map映射|Map]]操作，再做flat的操作； 
* 注意二：flatMap中的flat相当于深度为1；

```js
<script>
  const message = ["hello world", "你好 同学", "my name is axl"]
  const newMessage = message.flatMap(item => item.split(" "))
  console.log(newMessage)
  //['hello', 'world', '你好', '同学', 'my', 'name', 'is', 'axl']
</script>
```

# 二、Object fromEntries

那么如果我们有一个[[02 ES8特性|entries]]了，如何将其转换成对象呢？ 

* ES10提供了 Object.formEntries来完成转换：

```js
<script>
  const obj = {
    name: "why",
    age:18,
    height:1.88
  }
  
  //将对象转为数组
  const entries = Object.entries(obj)
  console.log(entries)
  //[Array(2), Array(2), Array(2)]
  
  //将数组在转为对象
  const info = Object.fromEntries(entries)
  console.log(info)
  //{name: 'why', age: 18, height: 1.88}
</script>
```
