# 一、认识解构
ES6中新增了一个从**数组或对象中方便获取数据**的方法，称之为解构Destructuring。 

* 解构赋值 是一种特殊的语法，它使我们可以将数组或对象“拆包”至一系列变量中。 

**我们可以划分为**：数组的解构和对象的解构。

- 数组的解构： 基本解构过程、顺序解构、解构出数组：…语法、默认值

```js
var [name1 = "aaa", name2, ...newNames] = names
```

- 对象的解构：基本解构过程、任意顺序、重命名、默认值

```js
var {name: varName = "why", ...rest} = object
```


# 二、解构的应用场景

**解构目前在开发中使用是非常多的**： 

* 比如在开发中拿到一个变量时，自动对其进行解构使用； 

```js
const { name, age } = infos
```

* 比如对函数的参数进行解构；

```js
function sum({ a, b }) {
	console.log( a +b )
}
sum( 10, 20 )
```
