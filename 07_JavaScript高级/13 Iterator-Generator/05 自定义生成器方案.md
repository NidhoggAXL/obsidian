# 一、生成器实现

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1732785626000ug49d3.png)

`yield*` 只可以在**生成器函数**中使用，当使用在类中的时候，只需要在 iterator protoc 协议**`[Symbol.iterator]`前面加一个`*`号** 就可以把这个函数变为一个生成器函数。

如果要得到这个生成器（迭代器）并使用

```js
//得到迭代器
const myIterator = student[Symbol.iterator]()
//使用迭代器
console.log(myIterator.next())
console.log(myIterator.next())
console.log(myIterator.next())
console.log(myIterator.next())
```
