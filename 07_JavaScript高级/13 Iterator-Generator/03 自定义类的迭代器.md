# 一、自定义类的迭代器
在前面我们看到[[05 数组Array|Array]]、String、[[08 数据结构-Set|Set]]、[[09 数据结构-Map映射|Map]]等类创建出来的对象都是可迭代对象： 

* 在面向对象开发中，我们可以**通过class定义一个自己的类，这个类可以创建很多的对象**： 
* 如果我们也希望自己的类创建出来的对象默认是可迭代的，那么在设计类的时候我们就可以**添加上 @@iterator 方法**；

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1734614706000icnphd.png)

```js
<script>
  class Person {
    constructor(name, age, friend) {
      this.name = name
      this.age = age
      this.friend = friend
    }

	//给friend增加迭代器
    [Symbol.iterator]() {
      let index = 0
      return {
        next: () => {
          if (index < this.friend.length) {
            return { done: false, value: this.friend[index++]}
          } else {
            return { done: true}
          }
        }
      }
    }
  }
  
  const student = new Person("axl", 18, ["abc", "mba"])
  console.log(student)
  //Person {name: 'axl', age: 18, friend: Array(2)}
  for (const item of student) {
 //对象里面只有friend增加了迭代器，所以只打印friend的值
    console.log(item)
    //abc
    //mba
  }
</script>
```

# 二、迭代器的中断
迭代器在某些情况下会在没有完全迭代的情况下中断：

* 比如遍历的过程中通过**break、return、throw**中断了循环操作； 
* 比如在解构的时候，没有解构所有的值；
