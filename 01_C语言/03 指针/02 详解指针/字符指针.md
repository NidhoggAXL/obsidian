还有一种使用的方法
```c
int main()
{
	char* p = "abcdef";
	//这里不是把一个字符串放到 p 指针变量里面
	//"abcdef" 是一个常量字符串
	printf("%s\n",p);
	return 0;
}
本质是把字符串的首字符的地址放在了指针 p 中.

但是要注意的是 这里的话就不可以对字符串进行改变，例如
int main()
{
	char* p = "abcdef";
	*p = 'w';
	return 0;
}
这就是一个语法的错误了，会报错 Segmentation fault-断错误
访问非法内存会报的错误。
常量不可以改变
```
