在写代码的时候难免要把*数组*或者*指针*传给函数，拿函数的参数该如何设计呢？

# 一维数组传参
```c
void text1(int arr[])
{}
void text1(int arr[10])
{}
void text1(int* arr)
{}

void text2(int* arr[20])
{}
void text2(int* *arr)
{}

int main()
{
	int arr1[10] = {0};
	int* arr2[20] = {0};
	text1(arr);
	text(arr2);
	return 0;
}
```

# 二维数组
```c
void text(int arr[3]p[5])
{}
void text(int arr[][5])//只可以省略行，但是不可以省略列
{}

void text(int (*arr)[5])
{}

int main()
{
	int arr[3][5] = {0};
	text(arr);//二维数组传参
	return 0;
}
```

# 一级指针传参
```c
#include <stdio.h>
void print(int* p , int len)
{
	int  i = 0;
	for( i = 0 ; i < len ; i++)
	{
		printf("%d ",*(p+i));
	}
}
int main()
{
	int arr[] = { 1 , 2 , 3 };
	int* p = arr;
	int len = sizeof(arr)/sizeof(arr[9]);
	print(p,len);
	return 0;
}
```

> [!tip] 思考
> 当一个函数的参数部分为一级指针的时候，函数能接收什么参数?

比如：
```c
void text(int* p)
{}
int main()
{
	int a = 10
	int* p = &a;
	text(&a);
	text(p);
	return 0;
}
```

# 二级指针传参
```c
#include <stdio.h>
void text(int** pc)
{}
int main()
{
	int a = 10;
	int* p = &a;
	int** pa = &p;
	text(pa);
	text(&p);

	int* arr[5] = {0};//指针数组
	text(arr);
	return 0;
}
```

# 函数指针传参
存放函数地址的一个指针

函数指针类型的定义：
```c
void (*)()-函数指针类型
```

**打印函数名就是打印这个函数的地址：**
![[Pasted image 20230915100458.png]]

函数指针的使用：
```c
int Add(int x , int y )
{
	return x+y;
}
int main()

{
	int a = 2;
	int b = 3;
	int (*p)(int , int) = Add;//函数指针
	printf("%d\n",(*p)(int , int));
	return 0;
}
```

*不同函数定义的函数指针方式也不同：*
```c
void Add(char* str)
{
	printf("%s\n",str)
}
int main()
{
	void (*p)(char*) = Add;
	// p 就是一个函数指针，指针 p 指向函数 Add 的地址
	(*p)("haha");//对函数指针调用
	return 0;
}
```

(*(void(*)())0)();

解析：
把这个代码分开来，结果就可以看的出来
(* (void(*)()) 0) ();

把 0 强制类型转换成一个函数指针类型：该指针指向的函数时无参无返回值
void (*)()-0 就是一个函数的地址

调用 0 地址处的该函数 - (* (void(*)()) 0) ( )

代码2
void(*signal(int,void(**)(int)))(int);
void(*  signal ( int,void(**)(int) )    )(int);

解析：
signal是一个函数声明
signal函数的参数有2个，第一个是int。第二个是函数指针。
干函数指针指向的函数的参数是int，放回类型是void

signeal函数的反回类型也是一个函数指针：该函数指针指向的函数的参数是int，返回类型是void。
