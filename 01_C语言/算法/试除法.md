* 原理:就是当计算1~10之间的素数的时候，那中间的2~9来进行试除，如果结果的余数为0且这连个数相等就是余数。
	* 计算m~n之间的余数，那中间的m+1~n-1的数来试除，当m~n/m+1~n-1的余数为零且相等的时候这个数就是余数。
# 计算1~10之间的素数。
	代码：不能计算 1 本身，因为1本身就是素数
	
```c
#include <stdio.h>
int main()
{	
	int i = 0;
	int n = 0;
	printf("查找1~10的素数\n"); 
	printf("找到的素数有：");
	for(i=1;i<=10;i++)
	{
		int j = 0;
		for(j=2;j<i;j++)
		{
			if(i%j==0)
			{
				break ;
			}
		}
		if(i==j)
		{
			n++;
			printf("%d ",i);
		}
	}
	printf("\n");
	printf("素数总数有:%d",n);
	return 0;	
}
```

* 改进 **(可以把 1 本身这个素数计算进去)**
	* 原理：如果`16 = 2*8 = 4*4` 这里设16=i，则a||b(或者)中至少有一个数字<=开品方 i
	* 代码；
```c
#include <stdio.h>
#include <math.h>
int main()
{	
	int i = 0;
	int n = 0;
	printf("查找1~10的素数\n"); 
	printf("找到的素数有：");
	for(i=1;i<=10;i++)
	{
		int j = 0;
		//也可以把 j<=sqrt(i) 编写成  j<=i/2
		// sqrt(i) <= i/2 
		for(j=2;j<=sqrt(i);j++)
		{
			if(i%j==0)
			{
				break ;
			}
		}
		if(j>sqrt(i)) 
		{
			n++;
			printf("%d ",i);
		}
	}
	printf("\n");
	printf("素数总数有:%d",n);
	return 0;	
}
```
