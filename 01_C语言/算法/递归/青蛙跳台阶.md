前言

一只青蛙可以一次跳一级台阶，也可以一次跳两级台阶，如果青蛙要跳上n级台阶，共有多少钟跳法?

*动态规划:用上一步的结果，来快速计算得到下一步的结果*

青蛙跳台阶的方法有：

| 台阶 | 跳法 |
| ---- | ------ |
| 1    |1      |
| 2    | 2      |
| 3    | 3      |
| 4    | 5      |
| 5    | 8      |
| n    | (n-1)+(n-2)       |

思路：![[00_obsidian相关/Excalidraw/青蛙跳台阶|青蛙跳台阶]]

```c
#include<stdio.h>
int wa(int n)
{
	if (n == 1) return 1;
	if (n == 2) return 2;
	if (n >2)
	{
		return wa(n - 1) + wa(n - 2);
	}
}
int main()
{
	int n = 0;
	scanf("%d", &n);
	int len = wa(n);
	printf("有%d个台阶时有%d钟跳法\n",n, len);
	return 0;
}
```
