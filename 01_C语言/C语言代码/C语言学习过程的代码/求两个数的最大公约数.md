* 原理：利用余数的计算，当余数为0的时候就是找到最大公约数。
	* 设查找的两个数为 m 和 n ，当 m%n 为 0 的时候 n 为最大公约数。如果当 m%n 不为 0 的时候就有余数 r ，这是n赋值与 m ，r 赋值与 n 在进行运算，一直这样循环下去，知道 m%n 为 0 的时候。
* 例子：当m=24，n=18的时候m%n=1...6，这时不为0为6，r为6，n赋值与m=18，r赋值与n=6，在进行m%n=2...0,余数为0了，这样就确定n=6为最大公约数。
 
代码:
```c
#include <stdio.h>
int main()
{	
	int m = 0;
	int n = 0;
	int r = 0;
	scanf("%d%d",&m,&n); 
	while(m%n!=0)
	//这里也可以编写成m%n ,这样就代表了不为0（0代表假） 
	{
		r = m%n ;
		m = n ;
		n = r ;
	}
	printf("%d\n",n);
	return 0;
}
```
