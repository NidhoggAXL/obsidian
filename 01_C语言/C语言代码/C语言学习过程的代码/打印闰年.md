* 判断是否为闰年的方法（要进行两次判断）。
	* 能被4整除并且不能被100整除是闰年。
	* 能被400整除的是闰年。

# 计算1000~2000年的闰年

代码：
```c
#include <stdio.h>
int main()
{	
	int year = 0;
	int n = 0;
	printf("1000~2000年的闰年有：\n"); 
	for(year=1000;year<=2000;year++)
	{
		if(year%4==0&&year%100)
		{
			printf("%d ",year);
			n++;
		}
		else if(year%400==0)
		{
			printf("%d ",year);
			n++;
		}
	} 
	printf("\n");
	printf("存在闰年的年数：%d",n);
	return 0;
}
//判断闰年除了上面的编写方式还可以 if((year%4==0 && year%100!=0) || year%400==0) 来判断是否为闰年。
```
