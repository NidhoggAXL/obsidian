# 把字符串中所有的字母改写成该字母的下一个字母，最后一个字母 z 改写成字母 a。大字母仍为大写字母，小写字母仍为小写字母，其它的字符不变。
代码：
```c
#include <string.h>
#include <stdio.h>
#include <ctype.h>
#define N 81
原有的字符串为：“Mn.123xyZ”，调用该函数后，
串中的内容为：“No.123yzA”。
main( )
{ 
	 char a[N],*s;
	 printf ( "Enter a string(输入字符串) : " );
	 gets ( a );//字符串的输入
	 printf ( "The original string is(原字符串是) : " );
	 puts( a );//没有改变的字符串输出
	 s = a; 
	 while(*s)
	 {
	 if(*s=='z')
	 *s='a';
	 else if(*s=='Z')
	 *s='A';
	 else if(isalpha(*s)) 
	 *s+=1 ; 
	 s++; 
	 }
	 printf ( "The string after modified(修改后的字符串) : ");
	 puts ( a );//改变后字符串的输出
}
```
