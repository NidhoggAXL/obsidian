# 结构体类型的声明

**结构的基础知识：**
> 结构是一些值的集合，这些值称为成员变量。结构的每个成员可以是不同类型的变量

结构的声明
```c
struct-结构体关键字
stu-结构体标签
struct stu-结构体类型

写法1
struct stu//定义一个结构体类型
{
	char name[20];//名字
	short age;//年龄
	char tele[12];//电话
	char sex[5];//性别
}s1,s2,s3;//特别注意这个;好不嫩少
//s1 s2 s3 是三个全局的结构体变量

写法2
typedef struct stu
{
	char name[20];//名字
	short age;//年龄
	char tele[12];//电话
	char sex[5];//性别
}stu;//对struct stu 重新命名为 stu
//stu-类型

int main()
{
	struct stu s1;//局部变量-对应写法1
	stu s2;//局部变量-对应写法2
	return 0;
}
```

**结构体类型**
> 结构的成员可以是标量、数组、指针!，甚至是其他结构体。

# 结构体变量的定义和初始化

**初始化**
```c
1.简单结构体变量的初始化
typedef struct stu
{
	char name[20];//名字
	short age;//年龄
	char tele[12];//电话
	char sex[5];//性别
}stu;

int main()
{
	//结构体变量的初始化
	stu s1 = {"张三",20,"123","男"};
	stu s2 = {"李四",30,"456","女"};
	return 0;
}

2.嵌套结构体变量的初始化
struct S
{
	int a;
	char c;
	char arr[20];
	double d;
};

struct T
{
	char ch[10];
	struct S s;
	char* pc;
}

int main()
{
	struct T t ={"123",{100,"hehe","hello world",3.14},NULL};
	printf("%s\n",t.s.arr);//hell0 world
	return 0;
}
```

# 结构体成员的访问
* 结构体变量访问成员，结构变量的成员是通过点操作符（.）访问的。点操作符接受两个操作数。例子参考本文的-嵌套结构体变量的初始化
* 结构体指针访问指向变量的成员有时候我们得到的不是一个结构体变量，而是指向一个结构体的指针。


# 结构体传参
```C
void print1(int* a)
{
	printf("%d",(*a));
}

void print2(int a)
{
	printf("%d",a)'
}

int main()
{
	int a = 10;
	print1(&a);
	print2(a);
	return 0;
}
```
那这里面居然都可以，那哪一个更加好呢？

答案是:首选print1函数。原因;
> 函数传参的时候，参数是需要压栈的。如果传递一个结构体对象的时候，结构体过大，参数压栈的的系统开销比较大，所以会导致性能的下降。
**自我理解的话就是：**
print2 使用的是实参传递给形参，而形参需要对实参进行一份临时的拷贝，当实参较大时，创建的哪一个形参也较大，会占用太多的内存。
print1 使用的是指针的方法，而指针做多也就创建一个字节的空间，这样可以节省内存。

**结论:结构体传参的时候，要传结构体的地址。**
