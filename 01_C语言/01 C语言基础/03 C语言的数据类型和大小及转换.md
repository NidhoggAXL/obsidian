# 库函数中的类型
* char//字符数据类型//**一个byte**
* short//短整型//**两个byte -32768->32767 **
	* short和int联合使用`short int n = 11;`
* int//整型 
	* 32位系统条件下占**4个byte -20亿->20亿 **
	* 64位系统条件下占**2个byte**
* long//长整型//**4 或 8 个 byte c (语言标准规定 sizeof（long）>sizeof (int）)**
* long  long//更长整型//**8 个byte**
* float//单精度浮点数//**4 个 byte**
	* `float e = 3.8f;`
* double//双精度浮点数//**32位机器上面是4个byte，64位机器上是8个byte**

add[5]={1,2,3,4,5};

# C语言存储大小和转换
## 单位大小转换
* 1pb=1024tb
* 1tb=1024gb
* 1gb=1024mb
* 1mb=1024kb
* 1kb=1024byte (字节)
* 1byte (字节)=8bit (比特)

## bit-比特
* 存储二进制数
	* 1bit 存储一个二进制（0 或者 1）总共8个。

## bye-字节
 * 一个字节等于 8 个 bit（比特）
	* 列如：1bit=01010101（8个二进制位）

## 类型的大小(单位字节)
#c语言的存储bit 
```c
 include <stdio. h>
int main()
{	printf ("%d\n", sizeof (char));//1个byt(字节)
	printf ("%d\n", sizeof (short));//2 个byte
	printf ("%d\n", sizeof (int));//4 个byte
	printf ("%d\n", sizeof (long));//4 或 8 个 byte c (语言标准规定 sizeof（long）>sizeof (int）)
	printf ("%d\n", sizeof (long long));//8 个byte
	printf ("%d\n", sizeof (float)); //4 个 byte
	printf ("%d\n", sizeof (double))；//8 个
	return 0;
}
```

*  这些不同的数据类型是为了更好的使用电脑中的储存空间。
*  这里的函数是可以包括的数据大小是：1byte 有 8 个 bit 则可以表达的最大数就是 2^8-1 这个数。
