# const
> [!tip] 定义
> const是常量的意思，用const修饰的变量只嫩读，不能修改它的值
> const可以修饰指针

**对变量(局部变量)更改为常量(全局变量)**
```C
int main()
{
	const int a = 10;//被const修饰，变为常量，不可以改变
	a = 20;
	return 0;
}
```
上面代码运行的话会报错。

# shutdown
* 命令提示符类型
	* ``shutdown -s`` 关机
	* ``shutdown -t 60（时间为60秒）`` 时间
	* ``shutdown -a`` 停止关机
