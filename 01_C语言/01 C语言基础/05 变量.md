# 在（{}）外的称为全局变量；反之称为局部变量。
**例如；**
```c
int a=1
int main ()
{
	int b=2
	return o ;
}
```
这里的 a 称为全局变量，b 称为局部变量。
* 局部变量在完成后会自动消除。


## 全局变量和局部变量可以相同，但是以局部变量优先。
**类如；**
```c
int a=1
int main ()
{
	int a=2;
	printf("%d",a);
	return o);
}
```
这是优先输出 a=2

> [!tip] 变量的生命周期指的是变量的创建到变量的销毁之间的一个时间段
> 局部变量的生命周期是︰进入作用域生命周期开始，出作用域生命周期结束。
> 全局变量的生命周期是∶整个程序的生命周期。

**作用域:**
1. 局部变量的作用域是变量所在的局部范围.//在“{}”内
2. 全局变量的作用域是整个工程。//在"int main"外的变量
