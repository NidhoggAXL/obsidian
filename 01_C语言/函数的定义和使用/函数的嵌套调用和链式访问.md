# 嵌套调用
```c
#include <stdio.h>
void new_line()
{
	printf("hahha");
}
void three_line()
{
	int i = 0;
	for(i = 0;i<3;i++)
	{
		new_line;
	}
}
int main()
{
	three_line();//main函数中对three_line调用，在three_line中对new_line调用。
	return 0;
}
```

# 链式访问
```c
int main()
{
	printf("%d\n",strlen("abc"));
	像这样把连个函数链接在一个语句中就是链式访问。
}
```
