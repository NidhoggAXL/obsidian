# 一、传入和获取参数

**方式一**:早期时，需要单独使用Loader-utils(webpack开发)的库来获取参数

**方式二**:目前，·已经可以直接通过this.getoptions()直接获取到参数

在使用loader时，传入参数。可以通过一个webpack官方提供的一个解析库 loader-utils，安装对应的库。

```shell
npm install loader-utils -D
```
