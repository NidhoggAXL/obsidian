# 一、搭建本地服务器

第一步：使用rollup-plugin-serve搭建服务

```shell
npm install rollup-plugin-serve -D
```

第二步：当文件发生变化时，自动刷新浏览器

```shell
npm install rollup-plugin-livereload -D
```

第三步：启动时，开启文件监听

```shell
npx rollup -c -w
```
