# 二、界面的基本搭建

观察表格的界面可以看到 **人民币符号（￥）** ,这里在开发中是会使用一个函数来进行数据的优化，放入到 **methods** 里面，因为不对 data 里面的数据进行操作。

```js
//forma格式价格
formaNum(num) {
 return '￥' + num
}
```

总价表格内容的要进行响应式的处理，就可以放到 **computed** 计算属性里面：这里使用JavaScript的高级属性 [[05 数组Array#8.3 reduce|reduce]]

```js
//计算价格总和
totalPrice() {
 return this.books.reduce((total, item) => {
  return total + item.price * item.count
 }, 0)
```

# 三、购买数量按钮

**分析**：最少肯定是购买一次，不能低于一次，那么当数量为1的时候这个按钮应该失效，使用[[01 高级元素的使用#^249908|disabled]]来实现。我更改数量的时候那么就需要知道是点击了哪一行的 tr ，这个时候就需要知道点击行的 index 来进行确认。

**增加数量的元素按钮和函数**：

```html
<button @click="increase(index)">+</button>
```

```js
//增加数量
increase(index) {
 this.books[index].count++
}
```

**减少数量的元素按钮和函数**：

```html
<button :disabled="item.count <=1 " @click="decrease(index)">-</button>
```

```js
//减少数量
decrease(index) {
 this.books[index].count--
}
```

# 四、删除按钮

删除也是要知道点击删除的到底是那一行的数据，所以也还是要知道 index 的，对 books 进行删除，这里要知道给到的 books 还是一个数组。

删除按钮的元素和函数：使用到的技术[[05 数组Array#二、数组的操作|splice]]

```html
<button @click="removeItem(index)">删除</button>
```

```js
// 删除书籍
removeItem(index) {
 this.books.splice(index, 1)
}
```

# 五、点击数据行显示背景

这里实现的目标是当数据发生点击的时候，就会添加一个 active 来动态的显示背景。

* 使用 v-bind 的缩写来改变 class 是否添加 active 的 [[04 绑定class和style#二、绑定class – 对象语法|class对象语法绑定]]
* 定义一个初始值 fromaIndex ，对当前 index 是否为 formaIndex 来进行检测
* 被点击（currentIndex）来改变初始 formaIndex 为当前点击的 index

**实现的按钮操作行的 tr ：**

```html
<tr :class="{active: index === formaIndex}"
@click="currentIndex(index)"
v-for="(item, index) in books"
:key="index">
```


**data 数据里面的 formaIndex 初始化：**

```js
data() {
	return {
	 books: books,
	 currentCount: false, //当前数量
	 formaIndex: -1//当前索引
	}
}
```

**按钮切换 formaIndex ：**

```js
//背景色变化
currentIndex(index) {
 //console.log(index)
 this.formaIndex = index
}
```
