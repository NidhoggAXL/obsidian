# 二、获取动态路由的值

那么在User中如何获取到对应的值呢？ 

* 在template中，直接通过 `$route.params`获取值； 
* 在created中(opsiton API)，通过 this.$route.params获取值； 
* 在setup中(composition API)，使用 vue-router 库给提供的一个hook <mark class="hltr-cyan">useRoute</mark>； 
	* 该Hook会返回一个Route对象，对象中保存着<mark class="hltr-cyan">当前路由</mark>相关的值；
	* **当前路由也被称为活跃路由**
	* 需要`import { useRoute } from 'vue-router'`导入

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/17617373720000b9ycs.png)

## useRoute

#useRoute和useRouter

**作用**：获取当前活动路由的信息（只读）

**返回值**：返回一个响应式的 `Route` 对象，包含当前路由的各种信息

**常用属性**：

- `path` - 当前路由路径
    
- `params` - 动态路由参数
    
- `query` - URL 查询参数
    
- `hash` - URL hash
    
- `name` - 路由名称
    
- `meta` - 路由元信息
    
- `fullPath` - 完整路径

```vue
<template>
  <div>
    <p>当前路径：{{ route.path }}</p>
    <p>路由参数：{{ route.params }}</p>
    <p>查询参数：{{ route.query }}</p>
  </div>
</template>

<script setup>
import { useRoute } from 'vue-router'

const route = useRoute()

// 监听路由参数变化
watch(() => route.params.id, (newId) => {
  console.log('ID 参数变化:', newId)
})
</script>
```

# 五、路由的嵌套

什么是路由的嵌套呢？ 

* 目前我们的Home、About、User等都属于第一层路由，我们在它们之间可以来回进行切换； 

但是呢，Home页面本身，也可能会在多个组件之间来回切换： 

* 比如Home中包括Product、Message，它们可以在Home内部来回切换；
* 这个时候就需要使用嵌套路由，在Home中也使用 router-view 来占位之后需要渲染的组件；
