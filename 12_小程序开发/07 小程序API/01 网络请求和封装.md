# 四、API封装

## 4.1 函数方式

### 4.1.1 封装

```js
export function xlRequest(config) {
  //wx.request不支持Promise的调用
  //我们自己把他封装在Promise里面就可以调用
  //防止回调地狱
  return new Promise((reslove, reject) => {
    wx.request({
      ...config,
      success: (res) => {
        reslove(res)
      },
      fail: (err) => {
        reject(err)
      }
    })
  })
}
```

### 4.1.2 使用

1. 基础的调用：

```js
// pages/04_learn_newWork/index.js
import { xlRequest } from "../../servers/index"
Page({
  data: {
    houseList: []
  },
  // 1.普通的网络请求调用
  onLoad() {
    xlRequest({
      url: "http://codercba.com:1888/api/home/houselist",
      data: {
        page: 1
      }
    }).then((res) => {
      this.setData({houseList: res.data.data})
    })
  }
})
```

> [!info] 缺点
> 调用的时候，如果需要对页面数据进行更改，这个时候还没有请求到数据，就可能会报错

2. [[07 异步函数|异步函数]]调用

```js
// pages/04_learn_newWork/index.js
import { xlRequest } from "../../servers/index"
Page({
  data: {
    houseList: []
  },
  //2.异步函数调用
  async onLoad() {
    const res = await xlRequest({
      url: "http://codercba.com:1888/api/home/houselist",
      data: {
        page: 1
      }
    })
    console.log(res);
    this.setData({ houseList: res.data.data })
  }
})
```
> [!info] 缺点
> 如果存在多个请求，那么只有等到第一个请求完成后才可以进行后面的请求。
> 数据过多的时候，会造成数据的阻塞

3. 把请求的过程在进行封装

```js
// pages/04_learn_newWork/index.js
import { xlRequest } from "../../servers/index"
Page({
  data: {
    houseList: []
  },
  onLoad() {
    //调用封装的请求函数
    this.getHouseList()
  },

  //封装一个异步请求函数
  async getHouseList() {
    const res = await xlRequest({
      url: "http://codercba.com:1888/api/home/houselist",
      data: {
        page: 1
      }
    })
    this.setData({ houseList: res.data.data })
  }
})
```

> [!info] 优点：
> - 请求的过程不在同一个异步函数中，不存在多个请求，需要等待
> - 分隔开来的异步请求，异步之间是不会阻塞的

## 4.2 类方式
 
### 4.2.1 封装

```js
class AXLRequest {

  constructor(BASE_URL) {
    this.baseURL = BASE_URL
  }

  request(config) {
    const { url } = config
    return new Promise((resolve, reject) => {
      wx.request({
        ...config,
        url: this.baseURL + url,
        success: (res) => {
          resolve(res)
        },
        fail(err) {
          reject(err)
        }
      })
    })
  }

  get(config) {
    return this.request({...config, method: 'get'})
  }

  post(config) {
    return this.request({...config, method: 'post'})
  }
}

export const AXLRequestInstance = new AXLRequest('http://codercba.com:1888/api')
```

### 4.2.2 使用

```js
// pages/04_learn_newWork/index.js
import { xlRequest, AXLRequestInstance } from "../../servers/index"
Page({
  data: {
    houseList: []
  },

  onLoad() {
    AXLRequestInstance.request({
      url: '/home/houselist',
      data: {
	      page: 1
      }
    }).then((res) => {
      console.log(res);
    })
  }
})
```
