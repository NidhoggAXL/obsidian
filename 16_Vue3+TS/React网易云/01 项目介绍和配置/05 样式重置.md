
安装重置css文件

```bash
npm install normalize.css
```

编写重置文件

```less title = "reset.less"
body, h1, h2, h3, h4, h5, ul, li, ol {
  padding: 0;
  margin: 0;
}

a {
  text-align: none;
  color: #333;
}

ul, li, ol {
  list-style: none;
}

img {
  vertical-align: top;
}

// input, button, select, textarea {
//   outline: none;
// }
```

编写通用css

```less title = "common.less"
body {
  font-size: 14px;
}
```

统一导出到出口文件

```less title = "index.less"
@import "./commin.less";
@import "./reset.less";

```

引入到项目的入口文件

```tsx title = "main.tsx"
// import { StrictMode } from 'react'
import { createRoot } from "react-dom/client"
import "normalize.css"
import "@/styles/global/index.less"

import App from "@/App"

createRoot(document.getElementById("root")!).render(<App></App>)

```
