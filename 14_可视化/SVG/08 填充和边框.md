# 一、填充和描边

如果想要给SVG中的元素上色，一般有两种方案可以实现： 

- 第一种：直接使用**元素的属性**，比如：**填充（fill）属性、描边（stroke）属性**等。 
- 第二种：直接编写**CSS样式**，因为SVG也是HTML中的元素，也支持用CSS的方式来编写样式。

# 二、元素的属性

直接使用元素的属性（填充和描边） 

- 在SVG中，绝大多数元素的上色都可通过 fill和stroke 两个属性来搞定。 
- fill属性：设置对象填充颜色。支持：颜色名、十六进制值、rgb、 rgba 、**currentColor（继承自身或父亲字体color）**。
- stroke属性：设置绘制对象的边框颜色。支持：颜色名、十六进制值、rgb、 rgba 。

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1758627042000h5m7oq.png)

## 2.1 填充属性(fill)

fill 填充属性，专门用来给SVG中的元素填充颜色。 

- **fill =“color”**。支持：颜色名、十六进制值、rgb、 rgba 、 **currentColor （继承自身或父亲字体color）**。 

如下图给矩形填充颜色：fill=“red”

```html
<svg width="400" height="400">
  <rect x="100" y="100" width="200" height="100" fill="red"></rect>
</svg>
```

![gh|300](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1758627200000i6xlku.png)

控制填充色的不透明：**fill-opacity = ”number ”**， 该属性专门用来控制填充色的不透明，值为 0 到 1。

```html
<svg width="400" height="400">
  <rect 
    x="100" 
    y="100" 
    width="200" 
    height="100" 
    fill="red"
    fill-opacity=".4"
  ></rect>
</svg>
```

## 2.2 描边属性(stroke)

stroke 描边属性 

- **stroke = “color”**： 指定元素边框填充颜色。 
- **stroke-opacity = “number”**：控制元素边框填充颜色的透明度。 
- **stroke-width = “number”**：指定边框的宽度。**注意，边框是以路径为中心线绘制的**。 
- stroke-linecap =“butt | square | round”：控制边框端点的样式，默认为 butt。 

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1758627965000obpj2u.png)

- stroke-linejoin = “miter | round | bevel”:控制两条线段连接处样式 ，默认为miter。

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1758627993000xfbnin.png)

- `stroke-dasharray ="number [, number , ….]"` : 将虚线类型应用在边框上。
	- 该值必须是用逗号分割的数字组成的数列，空格会被忽略。比如 10， 20 : 
	- 第一个表示填色区域长度为 10
	- 第二个表示非填色区域长度为 20

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1758628398000d3571d.png)

- stroke-dashoffset：指定在dasharray模式下路径的偏移量。 
	- 值为number类型，除了可以正值，也可以取负值。

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/17586285290006ibrh6.png)


# 三、CSS样式


直接编写CSS样式实现填充和描边 

- 除了定义元素的属性外，你也可以通过CSS来实现填充和描边（CSS样式可写在defs中，也可写在HTML头部或外部等）。 
- 语法和 HTML 里使用 CSS 一样，需要注意的是：**需要把 background-color、border 改成 fill 和 stroke** 
- **不是所有的属性都能用 CSS 来设置**，上色和填充的部分是可以用 CSS 来设置。 
	- 比如，fill，stroke，stroke-dasharray 等可以用CSS设置；
	- 比如，路径的命令则不能用 CSS 设置。

哪些属性可以使用CSS设置，哪些不能呢？

- SVG规范中将属性区分成 Presentation Attributes 和 Attributes 属性。 
- Presentation Attributes 属性( 支持CSS和元素用 )：https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/Presentation 
	- presentation - 展示
- Attributes 属性（只能在元素上作为属性用）： https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute 

>[!tip] 提示：这些属性是不需要去记的，用多了就记住了，在忘记时测试一下就知道了。

CSS给SVG中的元素填充、描边和上色，支持如下4种编写方式： 

- 方式一：内联（行内） CSS 样式，写在元素的style属性上 
- 方式二：内嵌（内部） CSS 样式，写在 [[07 SVG分组和复用#二、图形元素的复用(defs)|<defs>]] 中的 `<style>` 标签中
- 方式三：内嵌（内部） CSS 样式，写在 `<head>` 和 `<style>` 标签中
- 方式四：外部 CSS 样式文件，写在 `.css` 文件中\

>[!note] CSS样式优先级别：内联的style > defs中的style > 外部 / head内部 > 属性 fill
