# 一、axios的封装
`
```ts
import axios, {
  AxiosInstance,
  AxiosRequestConfig,
  AxiosResponse,
  AxiosError,
} from 'axios';

/**
 * HTTP 请求客户端类
 * 封装 axios 实例，提供 get、post 等常用方法
 */
class HttpClient {
  private instance: AxiosInstance;

  /**
   * 构造函数
   * @param baseURL - 基础 URL
   * @param defaultConfig - 默认请求配置
   */
  constructor(baseURL?: string, defaultConfig?: AxiosRequestConfig) {
    this.instance = axios.create({
      baseURL,
      timeout: 10000,
      ...defaultConfig,
    });

    // 可选：添加请求拦截器
    this.instance.interceptors.request.use(
      (config) => {
        // 可在此处添加 token 等逻辑
        return config;
      },
      (error) => Promise.reject(error)
    );

    // 可选：添加响应拦截器
    this.instance.interceptors.response.use(
      (response) => response,
      (error: AxiosError) => {
        // 统一错误处理（可自定义）
        console.error('Request failed:', error.message);
        return Promise.reject(error);
      }
    );
  }

  /**
   * GET 请求
   * @param url - 请求地址
   * @param config - 额外配置
   * @returns Promise<T> - 返回指定类型的数据
   */
  public get<T = any>(url: string, config?: AxiosRequestConfig): Promise<T> {
    return this.instance
      .get<T>(url, config)
      .then((res: AxiosResponse<T>) => res.data);
  }

  /**
   * POST 请求
   * @param url - 请求地址
   * @param data - 请求体数据
   * @param config - 额外配置
   * @returns Promise<T> - 返回指定类型的数据
   */
  public post<T = any>(
    url: string,
    data?: any,
    config?: AxiosRequestConfig
  ): Promise<T> {
    return this.instance
      .post<T>(url, data, config)
      .then((res: AxiosResponse<T>) => res.data);
  }

  /**
   * PUT 请求
   */
  public put<T = any>(
    url: string,
    data?: any,
    config?: AxiosRequestConfig
  ): Promise<T> {
    return this.instance
      .put<T>(url, data, config)
      .then((res: AxiosResponse<T>) => res.data);
  }

  /**
   * DELETE 请求
   */
  public delete<T = any>(url: string, config?: AxiosRequestConfig): Promise<T> {
    return this.instance
      .delete<T>(url, config)
      .then((res: AxiosResponse<T>) => res.data);
  }

  /**
   * PATCH 请求
   */
  public patch<T = any>(
    url: string,
    data?: any,
    config?: AxiosRequestConfig
  ): Promise<T> {
    return this.instance
      .patch<T>(url, data, config)
      .then((res: AxiosResponse<T>) => res.data);
  }
}

export default HttpClient;
```



使用实例：

```ts
import HttpClient from './HttpClient';

// 创建实例
const http = new HttpClient('https://jsonplaceholder.typicode.com');

// 定义响应数据类型
interface User {
  id: number;
  name: string;
  email: string;
}

interface Post {
  userId: number;
  id: number;
  title: string;
  body: string;
}

// GET 请求，类型推断自动生效
http.get<User>('/users/1').then((user) => {
  console.log(user.name); // ✅ 类型为 string
  console.log(user.email); // ✅ 类型为 string
});

// POST 请求
const newPost: Omit<Post, 'id'> = {
  userId: 1,
  title: 'foo',
  body: 'bar',
};

http.post<Post>('/posts', newPost).then((post) => {
  console.log(post.id); // ✅ 类型为 number
});

// 错误处理
http.get<User>('/users/999').catch((error) => {
  // error 为 AxiosError 类型
  console.error(error.message);
});
```

## 实现要点说明

1. **泛型支持类型推断**  
    每个方法都定义了泛型 `T`（默认 `any`），并通过 `instance.get<T>` 将响应数据的类型传递给 axios。  
    返回时直接取出 `res.data`，因此返回类型为 `Promise<T>`，调用者可以通过泛型指定期望的数据类型，TypeScript 会进行完整类型检查。
    
2. **Promise 返回值**  
    所有方法均返回 `Promise<T>`，支持 `async/await` 或 `.then/.catch` 链式调用。
    
3. **封装 axios 实例**  
    使用 `axios.create` 创建独立实例，可统一配置 `baseURL`、超时、拦截器等，避免全局配置干扰。
    
4. **错误处理**  
    通过响应拦截器统一捕获网络错误或 HTTP 状态码错误，并继续抛出，由调用方自行处理。
    
5. **可扩展性**  
    类结构清晰，便于添加更多 HTTP 方法（如 `put`、`patch`）或自定义拦截逻辑。
