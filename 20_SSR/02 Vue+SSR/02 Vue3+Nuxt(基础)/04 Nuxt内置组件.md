# 一、SEO组件

SEO组件： Html、Body、Head、Title、Meta、Style、Link、NoScript、Base

就是html元素提供的一些html类的元素

# 二、NuxtWelcome

NuxtWelcome：欢迎页面组件，该组件是 @nuxt/ui的一部分

# 三、NuxtLayout

NuxtLayout：是 Nuxt 自带的页面布局组件

# 四、NuxtPage

NuxtPage：是 Nuxt 自带的页面占位组件

 - 需要显示位于目录中的顶级或嵌套页面 pages/
 - 是对 router-view 的封装

# 五、ClientOnly

ClientOnly：

 - 该组件中的默认插槽的内容只在客户端渲染
 - 而fallback插槽的内容只在服务器端渲染

# 六、NuxtLink

NuxtLink：是 Nuxt 自带的页面导航组件 ，是 Vue Router `<RouterLink` 组件 和 HTML `<a>` 标签的封装。
