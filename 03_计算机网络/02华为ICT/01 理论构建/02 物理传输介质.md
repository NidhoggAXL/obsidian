# 一、双绞线
双绞线俗称：网线、跳线
* 跳线做名词就是网线
* 做动词就是连接网线

双绞线：
* 点信号：高电位为 1 ，低电位为 0
* 介质是铜（资源缺少）
* 传出距离短

**简单网络：**
![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17153260570006pval7.png)

* 两个终端，用一条能承载数据传输的物理介质(也称为传输介质)连接起来，就组成了一个最简单的网络。

## 1.1 双绞线类别
![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17153262610001ciyfl.png)

* 两根为一对
* 从上到下：10MB、100MB、1G

## 1.3 网线制作标准
**T568B：**
![[T568B]]
* 1、2、3、6 来进行一个数据的传输
* 4、5、7、8 是没有使用的，但是可以给电话用
* 1、2 用于数据的输出
	* 点信号用高低电平，所以有连个输入和输出
* 3、6 用于数据的输入

**T568A（线序）：**
把T568B的  1 和 3 交换、2 和 6 交换就是T568A的线序
![[T568A]]
* 这样 T568B 的1、2的输出就对应 T568A 的1、2的输入
* 这样 T568B 的3、6的输出就对应 T568A 的3、6的输入

> [!tip] 总结
> * **白橙、橙**组成输出，**白绿、绿**组成输入
> * 两台电脑互连就使用这钟线，两个电脑的线程不同
> 	* 这种线叫做**交叉线**
> 	* **设备连接的本质是连接网卡**
> * **同种设备**连接都使用交叉线（线序不一样）
> * **异类设备**连接都使用直通线（线序一样）

**什么是同种设备呢？**
区别不同设备的是使用3层级来区分的，3层以上的设备是同种设备，3层一下的是同种设备
* 三层以上设备：电脑、防火墙、路由器
	* 都可以配置 ip
* 三层一下设备：交换机
	* 交换机与交换机也是同种设备

## 1.4 实际使用中
在现在的使用中已经不需要在使用交叉线了，现在的网卡都很智能，会进行自我的适配。

在有一些的公司会把网线多余的四根线也利用起来，压上水晶头连接pc或者路由器，这样一条网线就可以连接4台设备。
但是不建议这样，有的商家会制成 4 根是铜线，4 根是铁线。

### 2.1.1 单模光纤和多模光纤的区别：

**传输距离：**

- 单模光纤：传输距离更远，可达数十公里甚至数百公里。
- 多模光纤：传输距离较短，通常在几百米到几公里范围内。

**光的波长：**

- 单模光纤：使用单一模式的激光光，波长较长（通常为 1310nm 或 1550nm）。
- 多模光纤：使用多模激光或 LED 光源，波长较短（通常为 850nm 或 1300nm）。

**选择区别：**

- **传输距离较远时：** 选择单模光纤。
- **传输距离较短时：** 可以使用单模或多模光纤，但多模光纤成本更低。
- **需要更高带宽时：** 选择单模光纤，因为它支持更宽的带宽。
- **成本考虑：** 多模光纤的成本通常低于单模光纤。
- **光源类型：** 单模光纤需要使用激光光源，而多模光纤可以使用激光光源或 LED 光源。
