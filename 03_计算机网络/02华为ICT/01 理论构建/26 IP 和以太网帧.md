# 一、IP
2000:/3为IPv6全球单播地址前缀，所以2000::或者3000::开头的地址都为lPv6全球单播地址。FF开头为组播,FE80为链路本地地址

## 2.1 IP 头部常见协议号
**传输层**

- TCP（传输控制协议）：6
- UDP（用户数据报协议）：17
- SCTP（流控制传输协议）：132

**互联网层**

- ICMP（互联网控制报文协议）：1
- IGMP（互联网组管理协议）：2
- IP-in-IP（IP 封装）：4
- IPv6（互联网协议第 6 版）：41
- GRE（通用路由封装）：47
- ESP（封装安全负载）：50
- AH（身份验证报头）：51

**应用层**

- FTP（文件传输协议）：21
- SSH（安全 shell）：22
- Telnet：23
- SMTP（简单邮件传输协议）：25
- DNS（域名系统）：53
- DHCP（动态主机配置协议）：67
- TFTP（简单文件传输协议）：69
- HTTP（超文本传输协议）：80
- POP3（邮局协议第 3 版）：110
- NNTP（网络新闻传输协议）：119
- NTP（网络时间协议）：123
- IMAP（互联网邮件访问协议）：143
- SNMP（简单网络管理协议）：161
- BGP（边界网关协议）：179

注意：这些协议号是由互联网号码分配机构（IANA）分配的，用于在 IP 头部的协议字段中标识协议。
