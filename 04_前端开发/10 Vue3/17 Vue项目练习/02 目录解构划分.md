| 目录文件           | 存放的内容                                                  |
| -------------- | ------------------------------------------------------ |
| assets         | 存放一些资源：img、css、font网络字体、audio/video音频                  |
| <br>components | 通用的一些组件：<br>common-多个项目都会用到的组<br>content-当前项目多个页面公用的组件 |
| hooks          | 公共的代码逻辑                                                |
| mock           | 模拟一些数据                                                 |
| router         | 路由相关的                                                  |
| sevice         | 网络请求                                                   |
| store          | 状态管理                                                   |
| utils          | 工具                                                     |
| views          | 存放一些页面                                                 |
