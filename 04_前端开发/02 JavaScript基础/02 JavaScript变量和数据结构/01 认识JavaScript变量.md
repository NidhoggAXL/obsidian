# 一、变量
## 1.1 变量的命名格式
在JavaScript中如何命名一个变量呢?包含两部分:

* 变量的声明:在JavaScript中声明一个变量使用var关键字(variable单词的缩写)(后续学习ES6还有let、const声明方式)
* 变量的赋值:使用 = 给变量进行赋值:

```js
var a = 1
console.log(a)
```

其他格式和大多数编程语言一样可以分开操作和同时声明多个变量

## 1.2 变量的命名规范
变量命名规则:**必须遵守**

* 第一个字符必须是一个字母、下划线`(_)`或一个美元符号`($)`
* 其他字符可以是字母、下划线、美元符号或数字
* 不能使用关键字和保留字命名!
	* 关键字：在js中已经被作为一个编写好并且命名的API
	* 保留字：有一天可能成为关键字
	* 网站:https://developer.mozilla.ora/zh-CN/docs/web/iavascript/reference/lexical_grammal
* 变量严格区分大小写


变量命名规范:建议遵守

* 多个单词使用驼峰标识
* 赋值=两边都加上一个空格:
* 一条语句结束后加上分号; 也有很多人的习惯是不加,;口 变量应该做到见名知意，

# 1.3 变量的使用注意
注意一：如果一个变量未声明(declaration)就直接使用，那么会报错;

注意二：如果一个变量有声明，但是没有赋值，那么**默认值是undefined(类型)**

注意三：如果没有使用var声明变量也可以，但是不推荐(**事实上会被添加到window对象上**)
