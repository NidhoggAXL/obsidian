# tabbar

可以在官网上面看到这样一段代码

```css
height: calc(env(safe-area-inset-bottom) + 50px);
```

calc 是 CSS 的一个计算函数

env(safe-area-inset-bottom) 则是为了适配移动端的一段代码，就是有些移动端视觉视口的底部会存在和用户交互的一段高度，为了防止和这段高度进行重叠而编写的代码。

```html
<div class="tabbar">
	<div class="item">
		<img src="./img/tabbar_home_active.png" alt="">
	</div>
	<div class="item">
		<img src="./img/tabbar_sort.png" alt="">
	</div>
	<div class="item">
		<img src="./img/tabbar_shop.png" alt="">
	</div>
	<div class="item">
		<img src="./img/tabbar_profile.png" alt="">
	</div>
</div>
```

```less
.tabbar {
    position: fixed;
    bottom: 0;
    left: 0;
    right: 0;
    display: flex;
    height: calc(env(safe-area-inset-bottom) + 50px);
    border-top: 1px solid #ddd;
    background-color: #fff;
    .item {
        flex: 1;
        text-align: center;
        img {
            width: 48px;
        }
    }
}
```
