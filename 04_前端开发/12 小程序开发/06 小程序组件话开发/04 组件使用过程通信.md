# 二、传递数据-properties

**给组件传递数据**： 

- 大部分情况下，组件只负责布局和样式，内容是由使用组件的对象决定的； 
- 所以，我们经常需要从**外部传递数据给我们的组件**，让我们的组件来进行展示；

**支持的类型**： String、Number、Boolean、Object、Array、null（<mark class="hltr-orange">不限制类型</mark>）

**默认值**： 可以通过value设置默认值；


自定义组件的代码：

```html
<view class="title">{{title}}</view>
<view class="content">{{content}}</view>
```

```js
Component({
  properties: {
    title: {
      type: String,
      value: "组件默认标题"
    },
    content: {
      type: String,
      value: "组件默认内容"
    }
  }
})
```

向自定义组件的传递数据的组件代码：

```html
<section-info title="龙族" content="每一个人心里都有一个死小孩"/>
```

```json
{
  "usingComponents": {
    "section-info": "/components/section-info/section-info"
  }
}
```

# 三、传递样式-externalClasses
给组件传递样式：有时候，我们不希望将样式在组件内固定不变，而是外部可以决定样式。

这个时候，我们可以使用externalClasses属性:

1. 在Component对象中，定义externalClasses属性
	![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1753755592000xl20oz.png)
	
2. 在组件内的wxml中使用externalClasses属性中的class
	![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17537556190001m9d55.png)
	
3. 在页面中传入对应的class，并且给这个class设置样式
