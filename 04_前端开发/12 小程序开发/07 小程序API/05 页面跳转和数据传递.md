# 二、wx的API跳转方式

## 2.1 页面跳转 - navigateTo

wx.navigateTo(Object object) 

- 保留当前页面，跳转到**应用内的某个页面**； 
- 但是**不能跳到 tabbar 页面**；

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1753798883000de6dmr.png)

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1753798914000hh5bcf.png)

## 2.2 页面返回 - navigateBack

wx.navigateBack(Object object) 

- 关闭当前页面，**返回上一页面或多级页面。**

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1753798965000hspnu3.png)

## 2.3 页面跳转 - 数据传递

### 2.3.1 方式一:早期

如何在界面跳转过程中我们需要相互传递一些数据，应该如何完成呢？ 

- **首页 -> 详情页**：使用URL中的query字段 
- **详情页 -> 首页**：在详情页内部通过[getCurrentPages](https://developers.weixin.qq.com/miniprogram/dev/reference/api/getCurrentPages.html)拿到首页的页面对象，直接修改数据
