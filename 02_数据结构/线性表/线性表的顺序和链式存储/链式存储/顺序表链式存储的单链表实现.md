## 复杂函数的编码
首先要知道一定，这里进行传参时要和顺序表有所不同，这里在结构体的初始化的时候是``SLTNode* plist = NULL``这里的实参是一个指针，要改变已经指针就需要使用二级指针来接受一级指针的地址，这样才可以改变一级指针-形参。
![[多级指针]]

* 尾插入
```c
void SListPushBack(SLTNode** pphead, SLTDateType x)//尾插入
{
	SLTNode* newnode = BuyListNode(x);
	if (*pphead == NULL)
	{
		*pphead = newnode;
	}
	else
	{
		SLTNode* tail = *pphead;//tail尾节点
		while (tail->next != NULL)
		{
			tail = tail->next;
		}
		tail->next = newnode;
	}
}
```

## 利用返回值来编码
* 查找(这里利用放回值的办法来查找)
```c
SLTNode* SListFind(SLTNode* phead, SLTDateType x)//查找
{
	SLTNode* cur = phead;
	while (cur->next != NULL)
	{
		if (cur->date == x)
		{
			return cur;
		}
		cur = cur->next;
	}
	return -1;
}
没有找到时，返回-1可以进行判断，找到时返回为x的结构体cur
```

* 删除pos后面的元素
```c
void SListEraseAftet(SLTNode* pos)//删除pos后面的一个元素
{
	assert(pos->next);
	//如果只有一个元素，则后面没有元素了，这里进行一个判断
	SLTNode* next = pos->next;
	pos->next = next->next;
	free(next);
}
```

* 在pos后面插入一个节点

* 在pos前插入一个节点
```c
void SListInsert(SLTNode** pphead, SLTNode* pos, SLTDateType x)//在pos前位置插入一个节点
{
	SLTNode* newnode = BuyListNode(x);
	if (pos == *pphead)
	{
		newnode->next = *pphead;
		*pphead = newnode;
	}
	else
	{
		SLTNode* cur = *pphead;
		while (cur->next = pos)
		{
			cur = cur->next;
		}
		SLTNode* newnode = BuyListNode(x);
		cur->next = newnode;
		newnode->next = pos;
	}
}
```
这里需要注意的是下面的这个代码不可以改变顺序，改变顺序后：先把newnode定义为头，后面`newnode->next=*pphead`就是相当于`newnode->next=newnode`所以这里的话就形成一个newnode的自身循环。打印的时候就会一直在创建的newnode这里一直循环。
```c
	newnode->next = *pphead;
	*pphead = newnode;
```
