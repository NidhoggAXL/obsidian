# 顺序存储结构定义：
```
把逻辑上相邻的数据元素存储在物理上相邻的存储单元中的存储结构。

简而言之，逻辑上相邻，物理上也相邻
```

# 顺序表中元素存储位置的计算
 ```
 假设线性表的每个元素需要占 L 个存储单元，则第i个元素的存储位置是：LOC（ai）= LOC（a1）+ （i-1）* L
```
