## 向下调整算法
#算法
>[!tip] 建小堆
>前提：**左右子树都是小堆**
>从根节点开始，选出左右孩子中最小的那一个，跟父亲比较，如果比父亲小就替换。然后在继续下调，调到叶子节点就终止。

```C
void AdjustDown(int* a, int n, int root)//向下调整算法
{
	int parent = root;
	int child = parent * 2 + 1;//默认为左孩子
	while (child < n)//找到叶子节点结束
	{
		//1.选出左右孩子中小的那一个
		if (child+1<n && a[child + 1] < a[child])
		//要注意只有左孩子的时候是会越界的，所以要判断是否有有孩子
		{
			child += 1;
		}
		if (parent > child)
		{
			Swap(&a[parent], &a[child]);
			parent = child;
			child = parent * 2 + 1;
		}
		else
		{
			break;
		}
	}
}

void Swap(int* n, int* m)
{
	int tmp = *n;
	*n = *m;
	*m = tmp;
}


void PrintSort(int* a, int n)
{
	for (int i = 0; i < n; i++)
	{
		printf("%d ", a[i]);
	}
	printf("\n");
}


void HeapSort(int* a, int n)
{
	//child=parent*2+1
	//n-1为最后一个节点
	//(n-1-1)/2为最后节点的父亲(左-右孩子都是一样的)
	for (int i = (n - 1 - 1) / 2; i >= 0; i--)
	{
		AdjustDown(a, n, i);
	}
	
}
```

>[!tip] 大堆
>只需要把**向下调整算法**中的**小于号改为大于号**就好了

>[!tip] 建堆的时间复杂度
>while中的O(n)
