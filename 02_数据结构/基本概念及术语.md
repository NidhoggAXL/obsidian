1. 数据
```
数据是对客观事物的符号表示
```

2. 数据元素
```
数据元素是数据的基本单位，在计算机程序中通常中被作为一个整体考虑和处理。
```

3. 数据项
```
数据项是数据不可分隔的最小单位，相当于记录的"禹"或字段。
```

4. 数据对象
```
数据对象是指性质相同的数据元素的集合，是数据的一个子集。
```
