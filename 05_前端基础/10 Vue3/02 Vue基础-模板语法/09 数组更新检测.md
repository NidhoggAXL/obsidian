# 一、数组更新检测

Vue 将被侦听的数组的变更方法进行了包裹，所以它们也将**会触发视图更新**。

这些被包裹过的方法包括： 

* push() 
* pop() 
* shift() 
* unshift() 
* splice() 
* sort()
* reverse() 

替换数组的bian
* 上面的方法会**直接修改原来的数组**； 
* 但是某些方法**不会替换原来的数组**，而是会生成新的数组，比如 [[05 数组Array#八、数组的高阶方法#8.1 filter-过滤|filter()]]、[[05 数组Array#五、数组的方法-slice、concat、join|concat()和slice()]]、[[05 数组Array#8.2 map-映射|map()]]

> 当产生新的数组的时候，就不会触发视图的更新
> 如果也想更新，只能把新参数的数组，赋值给原数组，这样才会更新
