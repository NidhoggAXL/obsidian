# 一、对象数组转为数组对象
<mark class="hltr-orange">对象数组</mark>中的对象相同的key并且对应相同的value进行一个归类：

* 第一步：获取到<mark class="hltr-orange">对象数组</mark>中要进行分组对象中的key，作为<mark class="hltr-yellow">数组对象</mark>的key
* 第二步：把<mark class="hltr-orange">对象数组</mark>中相同key的数据添加到<mark class="hltr-yellow">数组对象</mark>的对应key的数组中

对象数组：

```js
const objectArray = [ 
    {
        a: '100',
        name: "axl"
    }, 
    {
        b: '200',
        name: "mba"
    }, 
    {
        c: '300',
        name: "cba"
    },
    {
        d: '400',
        name: "axl"
    }
]
```


进行转换的编码方式一：

```js
const arrayObject = {}
for (let item of objectArray) {
// 1. 为 数组对象 添加 对象数组 中的key为name
  arrayObject[item.name] = []
}
for (let item of objectArray) {
// 2. 为每一个 对象数组 中 key 相同的数组添加数据
  const valueArray = arrayObject[item.name]
  valueArray.push(item)
}
```

进行转换的编码方式二：

```js
// 1. 创建一个空对象
const arrayObject = {}

// 2. 遍历对象数组，获取每一个对象
for(let item of objectArray) {

// 3. 让数组对象获取并设置和对象数组相同的key
  let valueArray = arrayObject[item.name]
  
  // 4. 获取的key后，对象的value是undefined的，进行判断
  //    4.1 如果第一次为对象数组添加item.name 时valueArray肯定为undefined的
  //    4.2 如果已经存在item.name相同的key，那么进行后面的7的push操作
  if (!valueArray) {
  
  // 5. 手动定义value的值
    valueArray = []
    
    // 6. 把手动定义的值，添加到数组对象中
    arrayObject[item.enumPictureCategory] = valueArray
  }
  
  // 7. 把对象数组中的每一个对象，添加到数组对象的key中
  valueArray.push(item)
  //valueArray是同一个引用（赋值引用）
}
```
