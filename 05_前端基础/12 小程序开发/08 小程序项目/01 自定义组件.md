
# 一、导航栏
需要在自定义的页面，将 json 配置文件进行如下配置(coutom-自定义)

```json
{
"navigationStyle": "custom"
}
```

> [!warning] 页面情况
> - 内容会之间显示到 屏幕 的顶部
> 
> ![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2025/1754124397000htqt6b.png)
> - 这个时候会把手机的状态栏也会混乱，**需要计算手机的状态栏高度**
> 	- 因为不同的手机，状态栏高度不同
> 	- 使用小程序通过的 [wx.getWindowInfo](https://developers.weixin.qq.com/miniprogram/dev/api/base/system/wx.getWindowInfo.html) API，可以获取到手机的状态栏高度
> - 获取并设置一个 view 组件，让组件的高度为获取到手机状态栏的高度。


这个时候就可以自定义状态栏了，<mark class="hltr-blue">状态栏在手机端的高度统一都是44px</mark>，不管什么手机都是
