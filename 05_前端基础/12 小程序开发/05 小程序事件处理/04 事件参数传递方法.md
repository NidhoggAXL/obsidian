# 一、事件参数的传递
## 1.1 `data-*`来传递数据

当视图层发生事件时，某些情况需要事件携带一些参数到执行的函数中, 这个时候就可以通过data-属性来完成： 

- 格式：data-属性的名称 
- 获取：e.currentTarget.dataset.属性的名称

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1753688308000usgu54.png)

## 1.2 mark传递数据
mark传递数据的形式：

- 格式：`mark:属性名称`
- 获取：e.mark.属性的名称

> [!tip] 细节
> - 父子组件：把数据放到同一个mark对象
> 	- 点击父组件只会获取父组件定义的数据
> 	- 点击子组件会获取到所有父组件的数据
> - 兄弟组件：数据在不同的mark对象
> 	- 兄弟组件之间数据不会进行共享

```html
<view class="block" mark:name="axl" bind:tap="btnMark">
  <text class="block-item" mark:index="100">mark</text>
</view>
```

```js
Page({
  btnMark(event) {
    console.log(event.mark);
  }
})
```

点击 block 父组件，打印的内容`{name: "axl"}`

点击 block-item 子组件，打印的内容 `{index: "100", name: "axl"}` 
# 二、参数传递的案例练习

案例要求：编写一个 tab-control 的，要求点击后可以自动的添加点击元素的样式（实际时 class ）

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1753746644000487no5.png)

```html
<!--pages/tab-control/tab-control.wxml-->
<view class="tab-control">
  <block wx:for="{{data}}" wx:key="item">
    <view 
      class="item {{currentIndex === index ? 'active' : ''}}"
      bind:tap="itemClick"
      data-index="{{index}}"
    >
      {{item}}
    </view>
  </block>
</view>
```

```js
// pages/tab-control/tab-control.js
Page({
  data: {
    data: ["衣服", "裤子", "鞋子", "帽子"],
    currentIndex: 0
  },
  itemClick(event) {
    //event.currentTarget.dataset.index获取data-index定义的数据
    this.setData({currentIndex: event.currentTarget.dataset.index})
    console.log(this.data.currentIndex);
  }
})
```

```css
/* pages/tab-control/tab-control.wxss */
.tab-control {
  display: flex;
  justify-content: space-between;
  align-items: center;
  height: 100rpx;
}

.item {
  flex: 1;
  text-align: center;
  padding: 5px;
}

.item.active {
  color: red;
  border-bottom: 3px solid red;
}

```
