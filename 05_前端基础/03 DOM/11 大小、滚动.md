# 一、元素的大小和滚动

clientWidth：contentWith+padding(不包含滚动条)

clientHeight:：contentHeight+padding

clientTop：border-top的宽度

clientLeft：border-left的宽度


offsetWidth：元素完整的宽度

offsetHeight：元素完整的高度

**offsetLeft**：距离最近定位元素或者最近的table,td,th，body 元素的x

**offsetHeight**：距离最近定位元素或者最近的table,td,th，body 元素的y

scrollHeight：整个可滚动的区域高度

scrollWidth：整个可滚动的区域宽度

scrolTop：滚动部分的高度

### 1. **offsetTop**

- **含义**：元素顶部到其最近**定位父元素**（offsetParent）顶部的距离（单位：像素）
    
- **包含**：元素自身高度 + 上方元素高度 + 滚动距离
    
- **不包含**：父元素的边框、外边距
    
- **Vue 使用场景**：
    
```html
<template>
    <div ref="targetElement">内容</div>
</template>

<script>
export default {
    mounted() {
    const topPos = this.$refs.targetElement.offsetTop;
    console.log("元素顶部位置:", topPos); // 用于滚动定位或布局计算
    }
}
</script>
```

### 2. **clientTop**

- **含义**：元素**上边框的厚度**（单位：像素）
    
- **典型值**：CSS 中 `border-top-width` 的值（例如 `border: 3px solid black` 时 clientTop=3）
    
- **Vue 使用场景**：

```js
// 在计算布局时排除边框影响
const contentHeight = element.clientHeight - element.clientTop;
```

### 3. **offsetWidth**

- **含义**：元素**完整可视宽度**（包括内容、内边距、边框、垂直滚动条）
    
- **计算公式**：  
    `offsetWidth = width + padding-left + padding-right + border-left + border-right + 垂直滚动条宽度`
    
- **Vue 示例**：
    
```html
<div ref="box" :style="{ padding: '10px', border: '2px solid' }">
    内容
</div>

<script>
mounted() {
    console.log("总宽度:", this.$refs.box.offsetWidth); // 包含内边距+边框
}
</script>
```

### 4. **clientWidth**

- **含义**：元素**内容区域宽度**（仅含内容+内边距，不含边框/滚动条）
    
- **计算公式**：  
    `clientWidth = width + padding-left + padding-right - 垂直滚动条宽度`
    
- **Vue 使用场景**：
    
```html
// 获取内容区可用宽度（布局计算常用）
const usableWidth = element.clientWidth;
```
### 图示对比

| 属性              | 包含内容              | 不包含内容     |
| --------------- | ----------------- | --------- |
| **offsetTop**   | 到定位父元素顶部的距离       | 父元素边框/外边距 |
| **clientTop**   | 上边框厚度             | 内容/内边距    |
| **offsetWidth** | 内容+内边距+边框+垂直滚动条   | 外边距       |
| **clientWidth** | 内容+内边距（减去垂直滚动条宽度） | 边框/外边距    |

### 在 Vue 中的注意事项

1. **访问时机**：需在 `mounted` 或之后的生命周期钩子中访问（此时 DOM 已渲染）
    
2. **响应式更新**：直接获取的值**非响应式**，需在窗口变化时手动更新：

```js
mounted() {
    window.addEventListener('resize', this.handleResize);
},
methods: {
    handleResize() {
    this.width = this.$refs.element.clientWidth; // 重新获取值
    }
}
```
    
3. **滚动容器**：对于可滚动容器，使用 `scrollTop`（当前滚动位置）配合 `clientHeight`（可视区域高度）实现滚动逻辑。
    

### 使用建议

- **布局计算**：优先使用 `clientWidth/clientHeight`（内容区域尺寸）
    
- **精确定位**：使用 `offsetTop/offsetLeft` 获取相对位置
    
- **边框处理**：`clientTop/clientLeft` 常用于校正边框偏移量
# 二、window的大小、滚动

**window的width和height**

* innerWidth、innerHeight:获取window窗口的宽度和高度(包含滚动条)
* outerWidth、outerHeight:获取window窗囗的整个宽度和高度(包括调试工具、工具栏)
* documentElement.clientHeight、documentElement.clientWidth:获取html的宽度和高度(不包含滚动条)

**window的滚动位置:**

* scrollX:X轴滚动的位置(别名pageXOffset)
* scrollY:Y轴滚动的位置(别名pageYOffset)


**也有提供对应的滚动方法:**

* 方法 scrolBy(x,y):将页面滚动至 **相对于当前位置**的 (x,y)位置;
* 方法 scrollTo(pageX,pageY)将页面滚动至 **绝对坐标**;
