# 一、伪类概括

什么是伪类？
* Pseudo-classes:翻译过来是**伪类**
* 伪类是选择器的一种，它用于选择处于**特定状态的元素**

什么是元素的状态呢？
> 举一个超链接的例子：
> * 什么都没有操作的时候是一个状态
> * 当鼠标移动到超链接上面是一个状态
> * 点击超链接的时候是一个状态
> * 后面被点击过的超链接是一个状态

**常见的伪类：**
1. 动态伪类(dynamic pseudo-classes)
* :link、:visited、:hover（掌握）、:active、:focus

2. 目标伪类(target pseudo-classes)
*  :target

3. 语言伪类(language pseudo-classes)
* :lang( )

4. 元素状态伪类(Ulelement states pseudo-classes)
* :enabled、:disabled、:checked

5. 结构伪类(structural pseudo-classes)(后续学习)
* :nth-child()、:nth-last-child( )、:nth-of-type( )、:nth-last-of-type( )
* :first-child、:last-child、:first-of-type、:ast-of-type
* :root、:only-child、:only-of-type、:empty

6. 否定伪类(negation pseudo-classes)(后续学习)
* :not0)

所有的伪类: https://developer.mozilla.org/zh-CN/docs/Web/CSS/Pseudo-classes

# 二、 动态伪类（dynamic pseude-classes）

* a:link 未访问的链接
* a:visited 已访问的链接
* a:hover 鼠标挪动到链接上 **(重要)**
* a:active 激活的链接(鼠标在链接上长按住未松开)
* a:focus 元素聚焦（获取焦点，tab 移动到的地方）
	* 文本输入框一聚焦后，会发生变化
	* 这就是[[12 CSS的盒子模型（Box Model）#八、外轮廓-outline（了解）]]。


> [!tip] 注意事项
> * :hover必须放在:link和:visited后面才能完全生效
> * :active必须放在:hover后面才能完全生效
> * **所以建议的编写顺序是** :link、:visited、:focus、:hover、:active
> * 直接给a元素设置样式，相当于给a元素的所有动态伪类都设置了

**除了a元素:hover、:active也能用在其他元素上**

```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        /* 没有访问 */
        a:link {
            color: red;
        }
        /* 访问过后 */
        a:visited {
            color: blue;
        }
        /* 元素被预选 */
        a:hover {
            color: aqua;
        }
        /* 长按 */
        a:active {
            color: darkorange;
        }
    </style>
</head>
<body>
    <a href="http://www.baidu.com">百度一下</a>
</body>
</html>
```
