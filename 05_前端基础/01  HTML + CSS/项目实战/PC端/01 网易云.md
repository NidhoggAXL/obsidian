# 四、按钮的实现

网易云里面的按钮是一个比较难的。

他的按钮是使用一种[[01 知识补充#四、精灵图(CSS Sprite)|精灵图]]来作为背景的，但是他的这个精灵图有点特别

* 按钮两边都存在[[12 CSS的盒子模型（Box Model）#5.5 圆角 boder-radius|圆角]]，但是给出的精灵图会发现其中的背景图片是只有一个方向有圆角，另一个方向的圆角切除出来，并不是合在一起的。
* 实现的话就是使用两张背景图拼凑出一个按钮
* 有的网站的按钮给出的精灵图是两边都没有圆角

> [!tip] 上面这种精灵图的好处：
> 考虑到网页中会在多个地方使用到精灵图，这样的设计就可以保证当需要按钮的宽度改变的情况下，依然可以使用同一张精灵图。
> 使用的时候我们只需要对精灵图中的圆角发在合适的宽度，这样就可以组合成一个两边都有圆角啦。
