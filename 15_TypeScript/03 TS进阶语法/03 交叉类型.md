# 一、认识交叉类型

另外一种类型合并，就是交叉类型（Intersection Types）： 

- 交叉类似表示**需要满足多个类型的条件**； 
- 交叉类型使用 **& 符号**；

来看下面的交叉类型： 

- 表达的含义是number和string要同时满足； 
- 但是有同时满足是一个number又是一个string的值吗？其实是没有的，所以MyType其实是一个[[03 TS数据类型#五、never类型|never类型]]；

```ts
type MyType = number & string
//相当于
//type MyType = never
```

# 二、交叉类型的使用

开发中，我们进行交叉时，通常是对对象类型进行交叉的：

```ts
interface name {
  name: string;
}
interface running {
  running: () => void;
}

// 交叉类型
type person = name & running;
const person1: person = {
  name: "why",
  running: () => {},
}
```
