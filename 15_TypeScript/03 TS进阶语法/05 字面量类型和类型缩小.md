# 一、字面量类型

## 1.1 认识字面量类型

除了前面的类型之外，也可以使用字面量类型（literal types）：

```ts
let message: 'axl' = 'axl'

//message = 'hello'
//会报错，Type '"hello'is not assignable to type "axl"
```

那么这样做有什么意义呢？  默认情况下这么做是没有太大的意义的，但是我们可以将多个类型联合在一起；

```ts
type Alignment = 'left' | 'right' | 'center'
function changeAlign(align: Alignment) {
  console.log("修改方向", align)
}
changeAlign('left')
//修改方向 left
```

const info: {url: string, method: 'GET'} = {

const info = {
  url: 'https://www.baidu.com',

# 二、类型缩小

## 2.1 认识类型缩小

类型缩小的英文是 **Type Narrowing（也有人翻译成类型收窄）**

我们可以通过类似于 `typeof padding === "number"` 的判断语句，来改变TypeScript的执行路径；

在给定的执行路径中，我们可以**缩小比声明时更小的类型**，这个过程称之为 缩小（ Narrowing ）; 

而我们编写的 `typeof padding === "number"` 可以称之为 类**型保护（type guards）**； 

常见的类型保护有如下几种： 

- typeof 
- 平等缩小（比如`===`、`!==`） 
- instanceof
- in
- 等等...

## 2.2 typeof

在 TypeScript 中，检查返回的值typeof是一种类型保护： 因为 TypeScript 对如何typeof操作不同的值进行编码。

```ts
function handlePrint(item: string | number) {
  if (typeof item === 'string') {
    console.log(item.length)
  } else {
    console.log(item * 2)
  }  
}

handlePrint("axl")//3
handlePrint(2)//4
```

## 2.3 平等缩小

我们可以使用Switch或者相等的一些运算符来表达相等性（比如`===`,`!==`,`==`, `and0!=` ）

```ts
type Direction = 'left'| 'right' |'center'

function turnDirection(direction: Direction) {
  switch (direction) {
    case 'left':
      console.log("调用left方法")
      break
    case 'right':
      console.log("调用right方法")
      break
    case 'center':
      console.log("调用center方法")
      break
    default:
      console.log("调用默认方法")
  }
}
```

## 2.4 instanceof

avaScript 有一个运算符来检查一个值是否是另一个值的“实例”：

```ts
function printValue(data: string | Date) {
  if (data instanceof Date) {
    console.log(data.getTime())
  } else {
    console.log(data)
  }
}
```

## 2.5 in操作符

avascript 有一个运算符，用于确定对象是否具有带名称的属性：in运算符 

如果指定的属性在指定的对象或其原型链中，则 in 运算符返回true；

```ts
interface ISwim {
  swim: () => void
}
interface IRun {
  run: () => void
}

const flish: ISwim = {
  swim: () => console.log("flish swim")
}
const dog: IRun = {
  run: () => console.log("dog run")
}

function handleAction(action: ISwim | IRun) {
  if ('swim' in action) {
    action.swim()
  } else if ('run' in action) {
    action.run()
  }
}

handleAction(flish)//flish swim
handleAction(dog)//dog run
```
