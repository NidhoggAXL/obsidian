
# 一、默认参数

从ES6开始，JavaScript是支持默认参数的，TypeScript也是支持默认参数的：

```ts
function foo(x: number, y: number = 100) {
  console.log(x, y)
}

foo(10)
foo(10, 20)
```

使用默认参数的时候，默认参数的类型可以省略：

```ts
function foo(x: number, y = 100) {
  console.log(x, y)
}

foo(10)
foo(10, 20)

//会报错，推断的y为number类型，string类型不可以传入
// foo(10, '123')
```

这个时候y的类型其实可以**看作是 undefined 和 number 类型的联合**。

```ts
function foo(x: number, y = 100) {
  console.log(x, y)
}

foo(10)
foo(10, 20)
foo(10, undefined)
```
