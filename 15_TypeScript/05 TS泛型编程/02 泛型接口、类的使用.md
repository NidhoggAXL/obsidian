# 一、泛型接口

在定义接口的时候也可以使用泛型：

```ts
interface IPerson<T> {
  name: T;
  age: number;
  height: T;
  handleValue:(value: T) => void;
}

const p: IPerson<string> = {
  name: '张三',
  age: 18,
  height: '1.88',
  handleValue: (value: string) => {
    console.log(value)
  }
}
```

可以定义泛型接口的默认类型:

```ts
interface IPerson<T = string> {
  name: T;
  age: number;
  height: T;
  handleValue:(value: T) => void;
}
```

# 二、泛型类

也可以编写一个泛型类：

```ts
class Person<T> {
  constructor(public x: T, public y: T) {
    this.x = x
    this.y = y
  }

  print() {
    console.log(this.x, this.y)
  }
}

//显示传入泛型类型，在推导出p1的类型
const p1 = new Person<number>(1, 2)
//p1的类型 const p1: Person<number>

//显示的定义泛型的类型
const p3: Person<number> = new Person(1, 2)

//不传入泛型类型，推导泛型的类型
const p2 = new Person('1', '2')
//p2的类型 const p2: Person<string>
```
