# 一、波形
**关于波形:**

音频的波形图是根据音频在各个不同时间点上的增益水平(采样值)被绘制出来的，对我们基础地了解音频的大致段落结构、声音分布有着一定的指导性意义。

**下面是在 PR 中的图片：**

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1724510225000idgqqv.png)

* 两条波形对应的是左声道和右声道。
* 波形较高的地方喜欢叙述为波峰，较低的地方喜欢叙述为波谷。

# 二、音频

PR 中会有一个电平计：

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1724510496000gugmyy.png)


* 可以看到音频每一个阶段的状态
* 右侧的刻度数用来描述的是 dB分贝（dicebel）
	* 这里的分贝描述的是一个声音的增益水平，最大为值为 0 
* 两条数值也是左右声道，
* 增益越靠近 0 的时候就越清晰可辨，但是一旦超出了 0 ，就会造成一定程度的音频信息丢失，这种情况俗称为爆音
	* 当出现这种状况的时候就会在最上方出现两个红灯，来进行提示。

> [!tip] 小提示
> 在做音频处理时，绝大多数音频分贝数最好能在**-30dB**以上，以保证清晰可辨。
>比较舒适的区间范围是0~15dB。请确保你的音频峰值不要超过0dB,这样它的大多数部分503在播放时就是“安全”的。
