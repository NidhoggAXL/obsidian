# 一、认识Bootstrap
![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1741944692000ugwhwf.png)

Bootstrap 读音/'bu:tstrap/，是一个非常受欢迎的**前端框架**，官方网站将其描述为。

* 最流行的 HTML、CSS 和 JS **框架**，用于在 web 上开发**响应式**、**移动优先的项目**。(v3.x)
	* 响应式页面:页面布局会随着屏幕尺寸的变化而自动调整布局，作用是适配各个屏幕。
* Bootstrap是**功能强大、可扩展，且功能丰富**的前端工具包。(v5.x)
* Bootstrap底层是使用**Sass构建**，支持定制(Sass、Color、CSS variable.)。(v5.x)
* Bootstrap中的**网格系统**、**组件以及强大的JavaScript 插件**可以让我们快速搭建响应式网站。(v5.x)

简单的理解

* Bootstrap是**由HTML、CSS和avascript编写可复用代码的集合(包括全局样式、组件、插件等)**，它是一个前端框架，使用该框架能够**快速开发出响应的网站(即适配PC、平板和移动端的网站)**。
* Bootstrap可以让我们免去编写大量的 CESS代码(write less)，让我们更专注于网站业务逻辑的开发。
* Bootstrap是开源免费的，可以从GitHub直接拿到源码。
