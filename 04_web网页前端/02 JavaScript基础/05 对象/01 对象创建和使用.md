# 一、认识对象

在数据类型中我们提到还有一种特别的类型:对象类型，

* 对象类型涉及到JavaScript的各个方面，所以掌握对象类型非常重要
* 对象类型是一种存储键值对(key-value)的更复杂的数据类型，

对象类型可以使用`{}`来创建的复杂类型，里面包含的是键值对(“key: value”);

**键值对可以是属性和方法(在对象中的函数称之为方法，key称之为属性，value称之为属性值)**

其中key是字符串(也叫做属性名propertyname，ES6之后也可以是Symbol类型，后续学习);

* 这个字符串的引号可以省略

```js
var bar = {
	"name": "why"//不省略key的引号
	age: 19//省略
	"My name": "axl"//对字符串的时候不可以省略
}
```

其中value可以是任意类型，包括基本数据类型、函数类型、对象类型等;


两个属于: 函数vs方法

* 函数(function):如果在]avascript代码中通过function默认定义一个结构，称之为是函数.
* 方法(method): 如果将一个函数放到对象中，作为对象的一个属性，那么将这个函数称之为方法.


对象嵌套和函数方法：

```js
var bar = {
  name: "why",
  friend: {//对象
    age: 19
  },
  running: function() {//函数方法
    //把height属性绑定到running方法上面
    height: 19.8
  }
}
```

# 二、创建对象和使用对象
对象的创建方法有很多，包括三种:

* 对象字面量(Object Literal):通过{}
* new Object+动态添加属性
* new 其他类

```js
//1.对象字面量
var bar = {
	name: "why"
}

//2.new Object()
//Object构造函数
var bar = new Object()
bar.name = "why"

//3.new 其他类()
function Person(){}
var bar = new Person()
```

对象的使用过程包括如下操作:

```js
var bar = {
	name: "why",
	age: 18
}
```

* 访问对象的属性;

```js
console.log(bar.name)//why
```

* 修改对象的属性;

```js
bar.name = "kobe"
console.log(bar.name)//kobe
```

* 添加对象的属性;

```js
bar.height = 19.8
console.log(bar.height)//19.8
```

* 删除对象的属性;   delete 操作符

```js
delete bar.name
console(bar.name)//undefined
```
