开发中的运用场景：

* 防止变量的冲突，是变量在函数的局部变量中
* 对按钮的监听ES5之前使用，后面ES5之后就不适用了-**理解就好**

```html
<button class="btn">按钮1</button>
<button class="btn">按钮2</button>
<button class="btn">按钮3</button>

<script>
var btnEIs = document.querySelectorAll(".btn")
for(var i = 0;i < btnEIs.length ; i++) {
  var btn = btnEIs[i];
  (function(m) {
	btn.onclick = function() {
	  console.log(`按钮${m+1}发生了点击`)
	}
  })(i)
}
</script>
```

i 作为一个全局变量，在我们进行监听的时候就会一直是 3 这样的我们就不能监听到那个按钮了，解决的办法就是把 i 变成一个局部变量，那一个变量去接收 i 在循环中的每一个局部变量的值。
