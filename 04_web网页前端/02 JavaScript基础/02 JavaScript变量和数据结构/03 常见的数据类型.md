# 一、Number类型
number 类型代表整数和浮点数。

```js
var a = 1
var b = 1.22
```

数字number可以有很多操作，比如`乘法*、除法/、加法+、减法-`等等,


除了常规的数字，还包括所谓的“特殊数值(“specialnumeric values”)”也属于Number类型 **(了解)**

* **Infinity**:代表数学概念中的 无穷大∞，也可以表示-Infinity;
	* 比如 1/0 得到的就是无穷大;
* **NaN**:NaN 代表一个计算错误，它是一个错误的操作所得到的结果;
	* 比如 字符串和一个数字相乘;


在之前我们学习过进制的概念，数字类型也有其他的进制表示方法:

* 十进制(掌握)、十六进制、二进制、八进制(了解)

```js
<script>
  //十进制
  var a = 111
  
  //二进制
  var b = 0x111
  
  //八进制
  var c = 0o111

  //十六进制
  var d = 0b111
  
  console.log(a, b, c, d)//111 273 73 7
</script>
```

数字表示的范围:

* 最小正数值:Number.MIN VALUE，这个值为:5e-324，小于这个的数字会被转化为0
* 最大正数值:Number.MAX VALUE，这个值为:1.9976931348623157e+308

**isNaN(函数)**

* 用于判断是否不是一个数字。不是数字返回true，是数字返回false。

```js
var a = 1
console.log(isNaN(a))//true
//返回的是一个布尔类型
```

# 二、string类型

在开发中我们经常会有一些文本需要表示，这个时候我们会使用字符串String:

* 比如人的姓名:coderwhy。地址:广州市。简介:认真是一种可怕的力量;

JavaScript 中的字符串必须被括在引号里，有三种包含字符串的方式。

* 双引号:**"Hello”**
* 单引号:**'Hello'**
* 反引号:**`Hello`**

```js
//${变量/表达式}
var name = `coder${123}`
console.log(name)
//控制台日志：coder123
```

前后的引号类型必须一致:

* 如果在字符串里面本身包括单引号，可以使用双引号
* 如果在字符串里面本身包括双引号，可以使用单引号;

# 三、boolean类型
Boolean(布尔)类型用于表示真假:

* 比如是否毕业,是否有身份证.是否购买车票,是否成年人;
* 比如开发中，我们会判断一个账号是否登录、是否是管理员、是否具备某个权限、是否拥有某个英雄、皮肤等

在JavaScript里面：

* true 表示真
* false 表示假

```js
<script>
  console.log(Boolean(1))//true
  console.log(Boolean({}))//true
  console.log(Boolean([]))//true
  
  console.log(Boolean(0))//false
  console.log(Boolean(""))//false
  console.log(Boolean(null))//false
  console.log(Boolean(undefined))//false
</script>
```

# 四、Undefined类型
Undefined 类型只有一个值，就是**特殊值 undefined.**

* 如果我们声明一个变量，但是没有对其进行初始化时，它**默认就是undefined:**

下面代码一样的：

```js
var info
var info = underfined
```

这里有两个注意事项:

* 注意一:**最好**在变量定义的时候进行**初始化**，而不只是声明一个变量;
* 注意二:不要显示的将一个变量赋值为undefined
	* 如果变量刚开始什么都没有，我们可以**初始化为0、空字符串、null**等值;


# 五、Object类型
Object是对象的意思，Object 类型是一个特殊的类型，我们通常把它称为**引用类型或者复杂类型**

* 其他的数据类型我们通常称之为 “原始类型”，因为它们的值**包含一个单独的内容(字符串、数字或者其他)**
* Object往往可以表示一组数据，是其他数据的一个集合;
* 在JavaScript中我们可以使用 花括号仆的方式来**表示一个对象**;

```js
var user = {
  name:"AXL",
  age:18,
  height:18.8
}
console.log(user)
console.log(user.name)//拿到对象里面的name
```



# 六、null类型
Null 类型同样只有一个值，即特殊值 null。

* null类型通常用来表示**一个对象为空**，所以通常我们在**给一个对象进行初始化时**，会赋值为**nuIl**;

```js
var user = null
```

为什么不创建一个空对象呢？

```js
var user = {}
```

从内存的角度：虽然是一个空对象，但是还是会**占据内存**，而 null 就不存在这种情况

从使用角度：在后续的使用过程中，空对象会被**隐式转换**为 true，而 null 则是转换为 flase，**开发中希望是 false**。


> [!tip] null和undefined的关系:
> undefined通常只有在一个变量声明但是**未初始化**时，它的默认值是undefined才会用到;
> 并且我们**不推荐直接给一个变量赋值为undefined，所以很少主动来使用;**
> **null值非常常用**，当一个变量准备保存一个对象，但是这个**对象不确定时**，我们可以先赋值为null;
