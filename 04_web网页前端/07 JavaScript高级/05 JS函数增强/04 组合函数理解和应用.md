# 一、组合函数概念的理解
**组合（Compose）函数**是在JavaScript开发过程中一种对函数的**使用技巧、模式：**

* 比如我们现在需要对**某一个数据**进行**函数的调用**，执行**两个函数fn1和fn2，这两个函数是依次执行的；** 
* 那么如果每次我们都需要**进行两个函数的调用**，操作上就会显得重复； 
* 那么**是否可以将这两个函数组合起来，自动依次调用呢？** 
* 这个过程就是**对函数的组合**，我们称之为 **组合函数（Compose Function）**；


不适用组合函数：完成一个数字先加2在乘2

```js
<script>
  //函数一：数字 +2
  function add(x) {
    return x += 2
  }
  
  //函数二：数字 *2
  function double(x) {
    return x *= 2
  }

  console.log(double(add(1)))//6
</script>
```

使用组合函数：完成一个数字先加2在乘2

```js
<script>
  //函数一：数字 +2
  function add(x) {
    return x += 2
  }
  //函数二：数字 *2
  function double(x) {
    return x *= 2
  }

  function xlFt(...fns) {
    var fnsLength = fns.length
    
    //是否传入参数判断
    if (fnsLength <= 0) return
    //判断传入的参数是否为函数
    for (var i = 0; i < fnsLength; i++) {
      if (typeof fns[i] !== "function") {
        throw new Error(`index position ${i} must be function`)
      }
    }
  
    //返回新的函数
    return function(...rests) {
      //首先先执行第一个传入的参数（函数）
      //this的使用是把fns的this绑定到result上面（严谨性）
      //apply后面的参数在传入的以后一定要是一个数据，而rests剩余函数本身就是一个数组
      var result = fns[0].apply(this, rests)
      for(var i = 1; i < fnsLength; i++) {
        //把第一个函数的返回值传入到后面的函数
        //第一个函数的返回值类型是number，而apply后买参数要为数组
        //所以后面的要使用方括号[]来把 number 转变为数组
        result = fns[i].apply(this, [result])
      }
      //一定要返回result，不然
      return result
    }
  }

  var newFunction = xlFt(add, double)
  console.log(newFunction(1))//6
</script>
```
