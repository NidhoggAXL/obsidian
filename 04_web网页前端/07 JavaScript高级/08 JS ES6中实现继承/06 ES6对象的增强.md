# 一、对象字面量的增强
ES6中对 对象字面量 进行了增强，称之为 Enhanced object literals（增强对象字面量）。 

**字面量的增强主要包括下面几部分：** 

* 属性的简写：Property Shorthand 

```js
<script>
  var name = "axl"
  var age = 18
  
  var obj = {
    name,
    age
    //不使用属性增强的代码
    // name: name,
    // age: age
  }
  
  function foo() {
    var message = "hello"
    var info = "world"
    //不可以返回多个，就返回一个对象
    return {
      message,
      info      
      //不适用属性增强的代码
      // message: message,
      // info: info
    }
  }
</script>
```


* 方法的简写：Method Shorthand 

```js
<script>
  var obj = {
    name: "axl",
    //方法的简写
    running() {
      console.log(this)
    }
    //不使用简写
    // running: function() {
    //   console.log(this)
    // }
    //箭头函数不可以简写
  }
</script>
```

* 计算属性名：Computed Property Names

```js
<script>
  var key = "address" + " axl"
  var obj = {
    //计算属性名
    [key]: "coder"
  }
  console.log(obj)
  //address axl: "coder"
</script>
```

> **对象不能使用计算属性名**


```js
<script>
  const obj = {name: "axl"}
  const bar = {
    [obj]: "mba"
  }
  console.log(bar)
  console.log(bar.obj)
</script>
```
