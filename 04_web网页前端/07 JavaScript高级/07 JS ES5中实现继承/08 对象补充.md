# 一、对象的方法补充（理解）
**hasOwnProperty** 

* 对象是否有某一个属于自己的属性（**不是在原型上的属性**） 

**in/for in 操作符** 

* 判断某个属性是否在某个对象或者对象的原型上 

**instanceof** (对象和构造函数)

* 用于检测**构造函数（Person、Student类）的pototype**，是否出现在某个实例对象的原型链上 

**isPrototypeOf** （对象和对象）

* 用于检测**某个对象**，是否出现在**某个实例对象的原型链**上

```js
<script src="./js/inherit_utils.js"></script>
<script>
  var obj = {
    name: "axl",
    age: 19
  }

  var info = creatObject(obj)
  info.address = "中国"
  
  //hasOwnProperty
  console.log(info.hasOwnProperty("name"))//false
  console.log(info.hasOwnProperty("address"))//true

  //in
  console.log("name" in info)//true
  console.log("address" in info)//true
  
  //for in 操作
  //不仅遍历自己对象上面的属性，也包括原型原型对象上的属性
  for(var key in info) {
    console.log(key)
    //address name age
  }
  
  //instanceof
  function Person() {}
  function Student() {}
  inheritPrototype(Student, Person)
  var stu = new Student()
  console.log(stu instanceof Student)//true
  console.log(stu instanceof Array)//false
  
  //isPrototypeOf
  console.log(Student.prototype.isPrototypeOf(stu))//true
  console.log(Person.prototype.isPrototypeOf(stu))//true
  console.log(Object.prototype.isPrototypeOf(stu))//true
</script>
```
