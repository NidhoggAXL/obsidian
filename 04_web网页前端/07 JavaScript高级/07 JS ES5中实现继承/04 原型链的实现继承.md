# 三、原型链继承的弊端
但是目前有一个很大的弊端:某些属性其实是保存在p对象上的(studying明明是在Student函数的显式原型上面设置的，却保存到了p对象上面);

* 第一，我们通过**直接打印对象是看不到这个属性的;**
* 第二，这个属性**会被多个对象共享，如果这个对象是一个引用类型，那么就会造成问题**
* 第三，**不能给Person传递参数**(让每个stu有自己的属性)，因为这个对象是一次性创建的(没办法定制化);
