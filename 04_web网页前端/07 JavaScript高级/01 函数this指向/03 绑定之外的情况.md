# 一、忽略显示绑定
如果在显示绑定中，我们传入一个null或者undefined，那么这个显示绑定会被忽略，使用默认规则:

```js
  <script>
    function fool() {
      console.log(this)
    }
  
    fool.apply("aaa")
    //String{'aaa'} - 会进行隐式的包装
    //这叫包装类型
    fool.apply(null)//Window
    fool.apply(undefined)//Window
  </script>
```

# 二、间接函数引用

创建一个函数的 **间接引用**，这种情况使用默认绑定规则。

* 赋值(obj2.foo = obj1.foo)的结果是foo函数;
* foo函数被直接调用，那么是默认绑定;

```js
  <script>
    var obj1 = {
      name: "axl",
      foo: function() {
        console.log(this)
      }
    }
  
    var obj2 = {
      name: "why"
    };
  
    //正常使用
    // obj2.foo = obj1.foo
    // obj2.foo()//{name: 'why', foo: ƒ}
  
    //间接调用
    (obj2.foo = obj1.foo)()//Window
  </script>
```
