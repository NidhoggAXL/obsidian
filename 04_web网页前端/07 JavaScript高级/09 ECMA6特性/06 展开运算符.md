# 一、展开语法
展开语法(Spread syntax)： 

* 可以在函数调用/数组构造时，将数组表达式或者string在语法层面展开； 
* 还可以在构造**字面量对象时**, 将对象表达式按key-value的方式展开； 
* 可以进行展开语法的要**具有可迭代性**；

展开语法的场景： 

* 在**函数调用**时使用； 

```js
<script>
  function foo(name, ...other) {
    console.log(name, other)
    //axl [18, 'eatting']
  }
  foo("axl", 18, "eatting")
</script>
```

* 在**数组构造时**使用； 

```js
<script>
  const arr1 = [1, 2, 3]
  const arr = [...arr1, "eatting", "running"]
  console.log(arr)
  //[1, 2, 3, 'eatting', 'running']
</script>
```

* 在**构建对象字面量**时，也可以使用展开运算符，这个是在ES2018（ES9）中添加的新特性；

```js
<script>
  const obj = {
    name: "axl",
    age: 18
  }
  
  //不可以这样调用
  //在函数的调用时，用展开运算符，将对应的展开数据，进行迭代
  // console.log(...obj)
  
  const bar = {
  //在另一个对象里面进行构建对象字面量时才可以使用展开语法
    ...obj,
    height: 18.8,
    active: "eatting"
  }

  console.log(bar)
  //{name: 'axl', age: 18, height: 18.8, active: 'eatting'}
</script>
```

> [!tip] 可迭代对象：
> 数组、String、Arguments

> [!tip] 展开运算符也是一种浅拷贝
