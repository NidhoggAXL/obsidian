# path常见的API
从路径中获取信息 

* dirname：获取文件的父文件夹； 
* basename：获取文件名； 
* extname：获取文件扩展名

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17442013950009u7ti1.png)


**路径的拼接：path.join** 

* 如果我们希望将多个路径进行拼接，但是不同的操作系统可能使用的是不同的分隔符； 
* 这个时候我们可以使用path.join函数；

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1744201531000flttox.png)


**拼接绝对路径：path.resolve**  ^b867a8

* path.resolve() 方法会把一个路径或路径片段的序列解析为一个绝对路径； 
* 给定的路径的序列是从右往左被处理的，后面每个 path 被依次解析，**直到构造完成一个绝对路径**； 
* 如果在处理完所有给定path的段之后，还没有生成绝对路径，则使用当前工作目录； ![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1744201608000diotg8.png)
* 生成的路径被规范化并删除尾部斜杠，零长度path段被忽略； ![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1744201664000pe9hpy.png)

* 如果没有path传递段，`path.resolve()`将返回当前工作目录的绝对路径；![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1744201741000isz2bl.png)
