使用的是 Vue 的 geolocation 来进行一个获取：

> [!tip] 注意事项
> 虽然 Vue 提供了这么一个 AIP 来让我们可以进行获取，但是当在 PC 端使用的时候回存在不同的浏览器对这个 API 的处理是不同的。
> 
> <mark class="hltr-orange">geolocation API 是返回给我们一个 经纬度的。</mark>
> 
> * 谷歌的话是通过把获取到的 经纬度 发送给谷歌的服务器，让服务器来返回是哪一个地方
> 	* 国内的 PC 端如果不进行代理是访问不了谷歌的服务器
> 	* 手机端不进行代理也可以进行一个访问
> 
> * Edge浏览器实现呢又是通过获取当前电脑的 IP 地址来确定是哪一个地区
> 	* 这个就不存在需要使用 VPN
