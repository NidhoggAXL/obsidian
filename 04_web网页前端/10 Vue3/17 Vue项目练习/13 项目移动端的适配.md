# 二、单位的转换

在项目中单位的转换使用到工具来进行转换，不需要手动转换

首先安装一个插件：

```
npm install postcss-px-to-viewport -D
```

> 因为在[[02 package 配 置 文 件#3.4 所有类型的 scripts(脚本)|生产环境]]的时候没有使用到，只是在开发中用到，所以-D


在项目根目录下面创建 postcss.config.js 文件：

1. 如果是在 Common JS 里面使用：

```js
module.exports = {
  plugins: {
    'postcss-px-to-viewport': {
      viewportWidth: 375,
      selectorBlackList: ["favor"]
    }
  }
};
```

2. 如果是 ES 环境下使用

```js
export default {
  plugins: {
    // 你的插件配置（示例）
    'postcss-px-to-viewport': {
      viewportWidth: 375,
      selectorBlackList: ["favor"]
    }
  }
}
```

> [!tip] 这只是一个最基本的使用，可以添加其他属性来进行转换单位的时候设置
