# 二、创建Vue项目
方式一：Vue CLI 

* 基于webpack工具；
* 命令：vue create 

方式二：create-vue 

* 基于vite工具； 
* 命令：npm init vue@latest

项目配置: 

* 配置项目的icon 
* 配置项目的标题 
* 配置jsconfig.json

本次项目使用 vite 工具，并且选择 router 路由和 pinia 状态管理的配置。
