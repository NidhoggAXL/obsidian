安装 [[03 Moment.js、Day.js|dayjs]] 库的使用

utils工具文件中 formate-date.js 的编码内容:

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1749734190000tlm18t.png)

```js
import dayjs from "dayjs";

export function formatMouthDate(date) {
  return dayjs(date).format("MM月DD日")
}
```

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1749734215000cdbwuh.png)

> [!tip] 明天的编码
> 不可以随便的进行+1,必须是要获取到当前通过 new Date() 得到的数据,进行操作
> * 先通过 getDate() 得到当前为那一日,在当前日进行+1
> * 在通过 setDate() 来设置日,才可以保证不会存在每个月,日的不同造成错误
