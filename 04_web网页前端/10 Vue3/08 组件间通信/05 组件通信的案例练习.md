首相进行分析：

* 整体式再一个 App.vue 的组件里面
* 标题又是一个子组件
* 显示的内容更具子组件来显示内容

标题子组件根据 App.vue 的 data 来显示内容,而显示内容的子组件要根据标题子组件来显示内容.

```html
<template>
 <div class="tab-control">
  <template v-for="(item ,index) in titles" :key="index">
   <div 
    class="item" 
    :class="{active: currentIndex === index}"
    @click="tabItem(index)">
   {{ item }}
  </div>
  </template>
 </div>
</template>

<script>
export default {
 props: {
  titles: {
   type: Array,
   default: () => [],
  },
 },
 data() {
  return {
   //记录index来进行active的切换
   currentIndex: 0,
  };
 },
 methods: {
  tabItem(index) {
   this.currentIndex = index;
   // 触发tab-click事件，传递当前索引值
   this.$emit("tab-click", index);
  },
 },
}
</script>

<style scoped>
 .tab-control {
  display: flex;
  align-items: center;
  background-color: #f0f0f0;
 }

 .tab-control .item {
  flex: 1;
  padding: 10px;
  text-align: center;
 }

 .tab-control .item.active {
  border-bottom: 2px solid #42b983;
  color: #42b983;
 }
</style>
```

根组件更具子组件传递过来的index来显示内容是什么：

```html
<template>
 <div class="app">
  <tab-control 
   :titles="['衣服', '鞋子','裤子']"
   @tab-click="itemBtn"></tab-control>
 </div>
 <h1>{{ titles[currentIndex] }}</h1>
</template>

<script>
import TabControl from "./TabControl.vue";
export default {
 data() {
  return {
   titles: ["衣服列表", "鞋子列表", "裤子列表"],
   currentIndex: 0
  };
 },
 components: {
  TabControl,
 },
 methods: {
  // 点击tab-control的item时，触发这个方法
  itemBtn(index) {
   this.currentIndex = index;
  },
 },
}
</script>

<style scoped>

</style>
```
