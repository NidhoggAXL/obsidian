# 一、子组件传递给父组件
**什么情况下子组件需要传递内容到父组件呢？** 

* 当子组件有一些事件发生的时候，比如在组件中发生了<mark class="hltr-orange">点击，父组件需要切换内容</mark>
* 子组件有一些内容想要传递给父组件的时候； 


我们如何完成上面的操作呢？ 

* 首先，我们需要在子组件中定义好在某些情况下触发的事件名称； 
* 其次，在父组件中以[[03 父组件传递子组件|v-on的方式]]传入要监听的事件名称，并且绑定到对应的方法中 
* 最后，在子组件中发生某个事件的时候，根据事件名称触发对应的事件；

# 二、自定义事件的流程

案例要求：创建一个计算机的算式

* 在 App.vue 里面显示计数的多少
* 在 AddCount.vue 里面进行检测按钮,通过影响来增加 App.vue 里面的数据进行改变
* 在 SubCcount.vue 里面进行检测按钮,通过影响来增加 App.vue 里面的数据进行改变


**AddCount.vue 和 SubCount.vue 文件:**

* addCount() 里面传入参数,方便后面发出发射数据(emit)
* this.$emit 来发射数据到父组件
	* this 指定是本文件的发射,父组件可以通过this知道文件的来源
	* 第一个参数是发射的名称(可以自己定义)
	* 第二个参数是发射出去的数据

```html
<template>
 <div class="add-count">
  <button @click="addCount(1)">+1</button>
  <button @click="addCount(5)">+5</button>
  <button @click="addCount(10)">+10</button>
 </div>
</template>

<script>
export default {
 methods: {
  addCount(addCount) {
   this.$emit("add", addCount)
  }
 }
}
</script>

<style scoped>

</style>
```

```html
<template>
 <div class="sub-count">
  <button @click="subCount(1)">-1</button>
  <button @click="subCount(5)">-5</button>
  <button @click="subCount(10)">-10</button>
 </div>
</template>

<script>
export default {
 methods: {
  subCount(subCount) {
   this.$emit("sub", subCount)
  }
 }
}
</script>

<style scoped>

</style>
```


**App.vue 文件:**

* 通过[[03 v-bind绑定属性]]来检测
* 检测到了改变在通过后面定义的 addBtn 和 subBtn 来进行改变
	* 当然是可以传入参数的,但是在<mark class="hltr-orange">子组件里面已经默认传递了数据</mark>,可以省略
	* 如果再 App.vue 里面自定义参数,那么默认传递出来的数据就会被改变
* 后面再 methods 里面就可以<mark class="hltr-orange">默认调用子组件里面传递出来的数据</mark>

```html
<template>
 <div class="app">
  <h2>当前的数字是: {{ count }}</h2>
  <!-- 监听传入add属性 -->
  <add-count @add="addBtn"></add-count>
  <!-- 监听传入sub属性 -->
  <sub-count @sub="subBtn"></sub-count>
 </div>
</template>

<script>
import AddCount from "./AddCount";
import SubCount from "./SubCount";

export default {
 //局部组件的注册
 components: {
  SubCount,
  AddCount
 },
 data() {
  return {
   count: 0
  }
 },
 methods: {
  subBtn(subCount) {
   this.count -= subCount
  },
  addBtn(addCount) {
   this.count += addCount
  }
 }
}
</script>

<style scoped>

</style>
```
