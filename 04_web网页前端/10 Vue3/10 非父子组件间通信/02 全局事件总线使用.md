# 一、全局事件总线mitt库
Vue3从实例中移除了`$on`、`$off` 和 `$once` 方法，所以我们如果希望继续使用全局事件总线，要通过第三方的库： 

* Vue3官方有推荐一些库，**例如 mitt 或 tiny-emitter**； 
* 这里我们主要讲解一下 hy-event-store 的使用；
	* 这是一个王红远老师创建的库

首先，我们需要先安装这个库：

```
npm install hy-event-bus
```

其次，我们可以封装一个工具eventbus.js：
