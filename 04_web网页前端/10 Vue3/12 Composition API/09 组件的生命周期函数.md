我们前面说过 setup 可以用来替代 data 、 methods 、 computed 等等这些选项，也可以替代 <mark class="hltr-orange">生命周期钩子</mark>。 

那么setup中如何使用生命周期函数呢？ 

* 可以使用直接导入的 onX 函数注册生命周期钩子；

```js
import {onMounted, onUpdated, onUnmounted} from 'vue'
```
