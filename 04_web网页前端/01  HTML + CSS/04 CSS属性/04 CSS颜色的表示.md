# 二、RGB
RGB是一种色彩空间，通过R(red，红色) 、 G(green，绿色)、B(blue，蓝色）三原色来组成了不同的颜色

也就是通过调整这三个颜色不同的比例，可以组合成其他的颜色;

 ![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17115287020002qa7gs.png)

RGB各个原色的取值范围是0~255;
