在HMTL5之前，我们的网站分布层级通常包括哪些部分呢?

* header、nav、main、footer

但是这样做有一个弊端

* 我们往往**过多的使用div**, 通过id或class来区分元素:
* 对于浏览器来说这些**元素不够语义化**
* 对于搜索引擎来说,**不利于SEO的优化**

HTML5新增了语义化的元素:

* `<header>`:头部元素
* `<nav>`:导航元素
* `<section>`:定义文档某个区域的元素
* `<article>`:内容元素
* `<aside>`:侧边栏元素
* `<footer>`:尾部元素
