**分析思路：**

* 第一个区域和第二个区域之间的连接处是没有边框的
* 第二、三区域中的元素都设置上下右的边框，不设置左边边框
	* 后面在对重叠的边框进行处理

处理思路参考[[08 浮动-float属性#^float-boder|浮动中存在边框的思路及其设置]]

## 1.1 common
```css
.kl_wrapper {
    width: 1100px;
    margin: 0 auto;
}
```
