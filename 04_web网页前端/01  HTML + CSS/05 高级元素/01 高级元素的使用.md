# 一、列表元素
**列表的实现方式**

**事实上现在很多的列表功能采用了不同的方案来实现:**

* 方案一: 使用`div`元素**来实现(比如汽车之家,知乎上的很多列表)**
* 方案二:使用**列表元素**,使用元素语义化的方式实现;

**事实上现在很多的网站对于列表元素没有很强烈的偏好,更加不拘一格,按照自己的风格来布局:**

* 因是列表元素默认的CSS样式,让它用起来不是非常方便,
* 比如列表元素往往有很多的限制, `ul/ol`中只能存放`li`,` li`再存放其他元素,默认样式等;虽然我们可以通过重置来解决,但是我们更喜欢自由的`div`

**HTML提供了3组常用的用来展示列表的元素**

* 有序列表:`ol、li`
* 无序列表:`ul、li`
* 定义列表:`dl、dt、dd`

## 5.1 input
input 是一个**行内替换元素**（和 img 一样可以设置高度和宽度）

**input 元素常见的属性：**

type：input 的类型

* text:文本输入框(明文输入)
* password:文本输入框(密文输入)
* radio:单选框
* checkbox:复选框
* button:按钮
* reset:重置
* submit:提交表单数据给服务器
* file:文件上传
* readonly:只读
* disabled:禁用 ^249908
* checked:默认被选中
	* 只有当type为radio或checkbox时可用
* autofocus:当页面加载时，自动聚焦
* name:名字
	* 在提交数据给服务器时，可用于区分数据类型
* value:取值

```html
<input type="text" readonly>
<input type="text" disabled>
 等同于
 <input type="text" disabled="disabled">
 这种情况是布尔属性的设置情况
```

**type类型的其他取值和input的其他属性,查看文档:**

* https://developer.mozilla.org/zhCN/docs/Web/HTML/Element/Input

## 5.2 布尔属性（boolean attributes）
 常见的布尔属性有**disabled、checked、readonly、multiple、autofocus、selected**

**布尔属性可以没有属性值，写上属性名就代表使用这个属性**

* 如果要给布尔属性设值，值就是属性名本身

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1722856875000na84ef.png)

## 5.3 表单按钮
**表单可以实现按钮效果:**

* 普通按钮(type=button):使用value属性设置按钮文字
* 重置按钮(type=reset):重置它所属form的所有表单元素(包括input、textarea、select)
* 提交按钮(type=submit):提交它所属form的表单数据给服务器(包括input、textarea、select)

```html
<form action="">
        <input type="button" value="我是普通按钮">
        <input type="reset" value="我是重置按钮">
        <input type="submit" value="我是提交按钮">
</form>
```

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1722860175000pxe88l.png)

我们也可以通过 button 按钮来实现

```html
<form action="">

        <button type="button">我是普通按钮</button>

        <button type="reset">我是重置按钮</button>

        <button type="submit">我是提交按钮</button>

</form>
```

也是一样的效果。

> input 是单标签，button 是双标签。

## 5.4 input 和 label 的关系
label 元素一般跟 input 配合使用，用来表示 input 的标题

label 可以跟某个 input 绑定，点击 label 就可以激活对应的 input 变成选中

```html
<label for="跳转的id">
	lable的内容
	<input id="" type="">
</label>
```


> lable 里面的 for 属性值就是我们点击 lable 要跳转聚焦 id 的地方，所以聚焦的地方不一定要放到 lable 双标签里面，也可以是别的地方。
> 但是开发中一般吧 input 放到 lable 里面。
> **当然 lable 的内容也不会空，不然也没有点击进行跳转的地方。**

**开发中使用的场景：**

例如：当我们在输入用户名和密码的时候，希望点击用户名或者密码就聚焦到输入框中，这个时候就会用到 lable 和 input 一起的使用。

```html
<div>
	<label for="username">
		用户名：
		<input id="username" type="text">
	</label>
</div>
<div>
	<label for="password">
		密码：
		<input id="password" type="password">
	</label>
</div>
```

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1722861684000ugfg8n.png)

点击用户名或者密码就直接进入到输入框中，不止点击输入框才会进入输入的模式了。

## 5.5 input作为单选框 radio
**我们可以将 input 的 type 类型设置为 radio 变成单选框:**

* name 值相同的 radio 才具备单选功能，name 值相同的则互斥。

```html
<label for="male">

	男<input id="male" type="radio" name="sex" value="male">

</label>

<label for="female">

	女<input id="female" type="radio" name="sex" value="female">

</label>
```

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/17228629780003ymg1c.png)

* 只能选中一个，因为他们的 name 值相同。
* value 则是提交给服务器的值，这里注意并不是（男或者女）而是 value 的值交给服务器
	* 男或者女本事只是一个显示的文本，并不会提交给服务器。

如果进行了提交，代码如下：

```html
<form action="/abc">

	<label for="male">

		男<input id="male" type="radio" name="sex" value="male">

	</label>

	<label for="female">

		女<input id="female" type="radio" name="sex" value="female">

	</label>

	<button type="submit">提交数据</button>

</form>
```

![gh](https://raw.githubusercontent.com/AXLflechazoPN/Obsidian/main/2024/1722863280000hjehhg.png)

## 5.7 textarea 的使用
**textarea的常用属性:**

* cols:列数
* rows:行数

> 列数和行数可以理解为设置输入框的大小，但是这个大小又是可以进行拉伸放大和放小。

**缩放的CSS设置**

* 默认：resize：both
* 禁止缩放:resize: none;
* 水平缩放:resize: horizontal;
* 垂直缩放:resize: vertical;
* 水平垂直缩放:resize: both;

```html
<form action="info">

	<label for="info">

		用户信息<textarea name="info" id="info" rows="" cols=""></textarea>

	</label>

</form>
```

## 5.8 select 和 option 的使用
option是select的子元素，一个option代表一个选项

**select常用属性**

* multiple:可以多选（布尔属性）
	* 多选的时候要按住 ctrl 键进行多选
* size:显示多少项

**option常用属性**
* selected:默认被选中（布尔属性）

```html
<select name="" id="">

	<option value=""></option>

</select>
```

## 5.9 form 元素
**form通常作为表单元素的父元素:**

* form可以将整个表单作为一个整体来进行操作;
* 比如对整个表单进行重置;
* 比如对整个表单的数据进行提交;

**form常见的属性如下:**

action

* 用于提交表单数据的请求URL

method

* 请求方法(get和post)，默认是get

```html
<form action="" method="post"></form>
```

* `target="_self"` 当前页面跳转
* `target="_parent"` 当前父级页面跳转
* `target="_top"` 当前顶层页面跳转
